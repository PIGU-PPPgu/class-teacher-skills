# 龙虾 × 班主任：主动工作流方案（展示版）

## 一句话定位

**龙虾不是一个被动回答问题的助手，而是一个面向班主任场景的主动工作流系统。**

它的核心价值不是“能聊天”，而是：
- 能记住班级事务
- 能按时间主动提醒
- 能自动生成班务草稿
- 能在外发前先给班主任确认

---

## 一、为什么这件事值得做

班主任最消耗精力的，不是单个大任务，而是大量高频、零散、重复、容易漏的事务：
- 家长群通知
- 考试与校历提醒
- 评语与学生观察记录
- 作业未交、迟到、异常波动
- 家长会、班会、活动材料
- 日报、月报、班务总结

这些工作不难，但碎，且经常打断人。

所以真正有价值的不是“问一句答一句”，而是做成一套：

> **会记、会催、会生成、会等确认再发** 的班主任工作流。

---

## 二、核心场景地图

### 1. 定时提醒 / 定时推送
- 每周任务清单自动推送
- 每日 7:00 今日提醒
- 考试前 N 天复习提醒
- 假期作业打卡提醒
- 校历事件提前提醒
- 值日生轮换提醒
- 天气 / 穿搭 / 极端天气提醒

### 2. 班务材料自动生成
- 家长群消息润色 + 待确认发送
- 每月班务日志到期提醒 + 初稿生成
- 班会课主题建议 + 教案初稿
- 家长会全流程材料
- 运动会 / 春游 / 研学全套材料自动生成
- 新学期开学清单自动启动
- 考试全流程管家

### 3. 学生管理与家校沟通
- 评语平时采集、期末批量生成
- 学生异常主动预警（迟到、违纪、波动）
- 作业未交自动追踪 + 分层提醒
- 请假自动流转
- 任课老师协调记录与跟进
- 家长群争执预警 + 回复草稿
- 生日 / 特殊日期提醒

### 4. 分析与汇总
- 每日班级播报 / 班级日报
- 学生画像自动更新
- 班级趋势分析
- 综合素质评价初稿生成
- 班费记账与月底汇总

## 三、现阶段已经具备的能力

当前仓库已经整理好的核心 skill：

### `class-assistant`
这是执行型 skill，已经能覆盖：
- 家长通知文案生成
- 学生评语 / 寄语生成
- 班会发言稿准备
- 成绩分析
- 考勤报告
- 学生谈话记录模板
- 特殊学生跟踪
- 班级日程模板
- 家长会材料支持

它现在更像：

**班主任数字助理 + 模板库 + 脚本工具包**

### `class-teacher-market-scan`
这是说明型 skill，主要用于：
- 解释本机已有能力
- 说明市场上有没有成熟标准品
- 区分“当前能做什么”和“下一步该补什么”

---

## 四、现阶段还不能吹太满的部分

以下场景方向明确，但大部分还属于下一阶段：
- 钉钉 / 微信 / 飞书群自动定时发送
- 自动接入考勤、作业、天气、校历系统
- 自动识别群争执并介入
- 无人确认直接对外发消息
- 自动化全流程闭环执行

还需要补齐的层包括：
- 平台 API 接入
- 定时任务调度（cron）
- 数据结构设计
- 审核 / 确认机制
- 对外发送执行层

一句实话：

> **现在已经有工作流骨架，但还不是完整产品。**

## 五、其他人如何调用我们的 skill

### 方式 1：直接在支持 skill 的 agent 环境中调用
适合内部自己用。

方式很直接：
1. 加载 `class-assistant`
2. 用自然语言描述任务
3. 由 skill 提供模板、步骤、脚本和参考资料

示例：
- 帮我写一个春游家长通知
- 分析这次月考成绩
- 记录今天和某学生的谈话
- 给我出一个家长会提纲

### 方式 2：把 skill 当成模板包 / 脚本工具包
适合没有完整 agent 平台的人。

可以直接使用：
- `SKILL.md`
- `references/`
- `scripts/`
- `examples/`

### 方式 3：后续封装成主动工作流产品
适合接飞书 / 钉钉 / 企业微信 / cron。

调用链：
- 数据源
- 触发器
- skill 决策层
- 输出草稿
- 班主任确认
- 对外发送

## 七、最值得先做的 5 个闭环

建议优先做：

1. **家长群消息润色 + 确认发送**
2. **每日 / 每周提醒自动生成**
3. **评语平时采集，期末批量生成**
4. **作业未交 / 迟到异常提醒**
5. **班级日报 / 班务月报自动汇总**

原因：
- 高频
- 痛点清楚
- 价值直观
- 容易形成第一批可见成果

## 九、结论

这套东西真正值钱的地方，不是它会不会聊天。

而是它开始像一个 **班主任工作系统**：
- 知道什么该记
- 知道什么时候该提醒
- 知道什么能自动起草
- 知道哪些动作必须先问班主任一声

**光会回复，不值钱。会主动干活，才值钱。**
