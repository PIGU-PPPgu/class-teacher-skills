## 使用建议

1. **学期初**：填充整个学期的重要日期
2. **每周日**：规划下周工作重点
3. **每日早**：查看当天待办事项
4. **每月底**：回顾完成情况，调整下月计划
5. **定期备份**：防止数据丢失
