## 记录原则

1. **客观真实**：如实记录，不带主观偏见
2. **及时性**：谈话/事件后尽快记录
3. **保密性**：妥善保管，不随意传播
4. **连续性**：持续跟踪，形成完整档案
5. **建设性**：记录目的是为了帮助学生成长
