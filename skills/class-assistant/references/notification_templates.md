# 家长通知模板库

## 1. 日常通知类

### 收缴费用通知
```
各位家长好：

本学期[费用名称]共计[金额]元，请于[截止日期]前通过[缴费方式]完成缴纳。

缴费项目明细：
- [项目1]：[金额]元
- [项目2]：[金额]元

如有疑问请私信联系。

谢谢配合！
[班主任姓名]
[日期]
```

### 活动报名通知
```
亲爱的家长：

学校将组织[活动名称]，现开始报名。

活动时间：[日期] [时间]
活动地点：[地点]
参与对象：[年级/班级]
费用：[金额]元

报名方式：[如何报名]
报名截止：[截止日期]

本次活动[自愿/全员]参加，请根据实际情况报名。

[班主任姓名]
[日期]
```

### 调课/放假通知
```
各位家长：

接学校通知，[日期]因[原因]调课/放假安排如下：

- [原安排] → [新安排]
- 放学时间：[时间]

请家长提前做好安排，按时接送孩子。

[班主任姓名]
[日期]
```

## 2. 学习相关

### 作业提醒
```
家长您好：

近期发现部分同学[作业问题描述]，请家长协助督促：

1. 每日检查作业完成情况
2. 关注作业书写质量
3. 签字确认已完成

良好的作业习惯是成绩提升的基础，家校配合效果更佳。

感谢您的支持！
[班主任姓名]
```

### 考试通知
```
各位家长：

[考试名称]定于[日期]举行，具体安排：

科目及时间：
- [科目1]：[时间]
- [科目2]：[时间]

复习建议：
1. [建议1]
2. [建议2]
3. [建议3]

请家长协助孩子做好复习准备，保证充足睡眠。

[班主任姓名]
[日期]
```

### 成绩发布（私聊模板）
```
[家长姓名]您好：

[学生姓名]本次[考试名称]成绩如下：

总分：[分数]，班级排名：[名次]，年级排名：[名次]

各科成绩：
- [科目1]：[分数] [评价]
- [科目2]：[分数] [评价]

整体评价：
[进步/稳定/退步]表现，[具体表现描述]

建议：
1. [建议1]
2. [建议2]

欢迎沟通交流。
[班主任姓名]
```

## 3. 行为管理

### 表扬通知
```
[家长姓名]您好：

很高兴告诉您，[学生姓名]最近[具体表现]，值得表扬！

具体表现：
- [表现细节1]
- [表现细节2]

孩子的进步离不开家庭的支持，感谢您的教育配合！

[班主任姓名]
[日期]
```

### 问题提醒（温和版）
```
[家长姓名]您好：

想和您沟通一下[学生姓名]最近的情况：

[问题描述，具体事例]

建议我们一起：
1. [家校配合建议1]
2. [家校配合建议2]

孩子正处于成长期，有些反复是正常的。我们多关注、多引导，相信会好起来的。

期待您的回复，我们可以进一步沟通。

[班主任姓名]
[日期]
```

### 问题提醒（严肃版）
```
[家长姓名]您好：

关于[学生姓名]的[问题行为]，需要严肃沟通：

[具体问题描述]

这已经[次数]发生，影响了[具体影响]。

请家长高度重视，配合做好以下工作：
1. [要求1]
2. [要求2]

请于[时间]前与我联系，商讨后续教育方案。

[班主任姓名]
[日期]
```

## 4. 紧急情况

### 突发疾病/受伤
```
紧急通知：

[学生姓名]在[时间]发生[情况]，现已[处理措施]。

[症状描述]

建议：
1. [建议家长采取的行动]
2. [是否需要到校]

请家长尽快与我联系：[电话]

[班主任姓名]
[时间]
```

### 恶劣天气停课
```
紧急通知：

因[天气情况]，接教育局通知，明天([日期])停课一天。

安排如下：
1. [作业安排]
2. [线上学习安排]
3. [返校时间]

请家长做好孩子居家安全防护。

[班主任姓名]
[日期]
```

## 5. 特殊场合

### 家长会邀请
```
各位家长：

定于[日期] [时间]召开[学期/主题]家长会。

地点：[教室/会议室]
主要内容：
1. [内容1]
2. [内容2]
3. [内容3]

请准时参加，如有特殊情况请提前告知。

[班主任姓名]
[日期]
```

### 学期寄语/总结
```
亲爱的家长们：

[学期]即将结束，感谢您一学期以来的支持与配合！

本学期回顾：
- [班级整体表现]
- [重要活动回顾]
- [成绩总结]

假期建议：
1. [学习方面]
2. [生活方面]
3. [安全提醒]

祝大家假期愉快，我们[下学期/下学期]再见！

[班主任姓名]
[日期]
```

## 使用提示

- 根据实际情况调整语气和内容
- 重要通知建议私信发送
- 群通知注意保护学生隐私
- 敏感话题优先电话沟通
