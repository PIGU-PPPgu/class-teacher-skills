# 模板｜一对一家长沟通

## 模板 1｜日常问题反馈版
**适用：** 作业、课堂状态、行为习惯等日常沟通  
**语气：** 温和、合作式

```text
您好，想和您沟通一下孩子最近在学校的情况。

这段时间我观察到[具体表现]，主要体现在[问题/情境]。我已经在学校这边做了[已采取措施]，孩子也有一定的意识。

想请您这边也一起配合，提醒孩子[配合动作/建议]。如果家里这边也有相关情况，欢迎随时和我沟通，我们一起帮助孩子调整。

感谢您的理解与配合。
```

## 模板 2｜进步反馈版
**适用：** 向家长反馈积极变化  
**语气：** 正向、真诚

```text
您好，和您分享一下孩子最近在学校的一些进步。

最近孩子在[方面]有比较明显的提升，比如[具体表现]。看得出来他/她这段时间在认真调整，也值得肯定。

建议您在家里也可以多给予鼓励，帮助孩子把这样的状态继续保持下去。特此和您分享，也感谢您一直以来的配合。
```

## 模板 3｜电话沟通提纲版
**适用：** 先整理口头沟通逻辑  
**语气：** 简洁、清楚

```markdown
### 家长电话沟通提纲
1. 先说明来意：[沟通主题]
2. 客观反馈情况：[具体表现]
3. 说明学校已做措施：[已采取措施]
4. 希望家长配合：[建议]
5. 结束语：感谢配合，后续继续保持沟通
```

## 使用提示
- 优先写事实，不先下判断
- 如果要提建议，尽量具体到可执行动作
- 对家长的表达尽量保留合作空间
