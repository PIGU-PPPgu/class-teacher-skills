# 模板｜重点学生阶段观察卡

## 模板 1｜标准观察卡
**适用：** 连续跟踪重点学生阶段表现  
**语气：** 客观、跟进导向  
**可替换字段：**
- [学生]
- [观察周期]
- [当前关注点]
- [学习状态]
- [纪律表现]
- [情绪状态]
- [同伴互动]
- [家长反馈]
- [风险等级]
- [下一步动作]

```text
学生：[学生]
观察周期：[观察周期]
当前关注点：[当前关注点]
学习状态：[学习状态]
纪律表现：[纪律表现]
情绪状态：[情绪状态]
同伴互动：[同伴互动]
家长反馈：[家长反馈]
风险等级：[风险等级]
下一步动作：[下一步动作]
```

## 模板 2｜阶段摘要版
**适用：** 给内部复盘或交接使用  
**语气：** 概括、简洁

```text
[学生] 在 [观察周期] 内，主要关注点为 [当前关注点]。
阶段表现：
- 学习：[学习状态]
- 纪律：[纪律表现]
- 情绪：[情绪状态]
- 同伴互动：[同伴互动]
当前判断：[风险等级]
后续安排：[下一步动作]
```
