# 场景 29｜重点学生阶段观察卡

适合把重点学生在一个观察周期内的表现沉淀成固定字段观察卡。

## 这个场景解决什么问题
- 长期跟进只靠印象，不利于连续判断
- 需要把学习、纪律、情绪、同伴互动统一收口
- 需要支撑谈话、家校沟通、阶段复盘或内部交接
- 需要更稳定地识别风险等级和下一步动作

## 建议先看
- `SCENARIO.md`：场景说明、输入输出、边界
- `templates.md`：标准观察卡模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 阶段观察卡
- 风险等级判断草稿
- 下一步跟进建议

## 一句话用法
把观察周期内的关键事实、家长反馈和老师判断给 AI，让它整理成结构化观察卡。
