# 模板｜请假登记 / 请假说明

## 模板 1｜班主任留档版
**适用：** 内部登记、后续跟进  
**语气：** 客观、规范  
**可替换字段：**
- [学生姓名/标识]
- [请假日期]
- [请假时长]
- [请假原因]
- [返校时间]
- [备注]

```text
【请假登记】
学生：[学生姓名/标识]
请假日期：[请假日期]
请假时长：[请假时长]
请假原因：[请假原因]
预计返校时间：[返校时间]
备注：[备注]
```

## 模板 2｜科任老师同步版
**适用：** 向科任老师同步缺课情况  
**语气：** 简洁、明确  
**可替换字段：**
- [学生姓名/标识]
- [请假时间]
- [原因]
- [返校安排]

```text
各位老师好，[学生姓名/标识]因[原因]于[请假时间]请假，预计[返校安排]返校。
如有当日作业或课堂任务需要补充，我这边再统一跟进。谢谢。
```

## 模板 3｜家长确认回执版
**适用：** 与家长确认请假信息  
**语气：** 温和、规范  
**可替换字段：**
- [学生姓名]
- [请假时间]
- [返校时间]
- [提醒事项]

```text
家长您好，已收到[学生姓名]于[请假时间]的请假说明。
如返校时间有变化，请及时告知。另请留意[提醒事项]，谢谢配合。
```
