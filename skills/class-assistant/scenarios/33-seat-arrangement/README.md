# 场景 33｜座位编排 / 调座方案

适合根据学生名单、座位结构和班级规则，快速生成一版可执行的座位表与调座说明。

## 这个场景解决什么问题
- 只靠手工排座位很费时间
- 需要同时兼顾小组合作结构、学业层次、性格、品行、上课态度、学习态度等约束
- 常见排法包含按学业层次分号搭配，例如前 12 名为 1 号、中间梯队为 2 号 / 3 号、后 12 名为 4 号，并按 1-4、2-3 组合搭配同桌或组内协作
- 需要先出内部排位草稿，再决定是否发学生版
- 调座后还需要一份简明说明，方便班级执行

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：标准座位表与调座说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 座位表（按行列）
- 调座方案说明
- 特殊约束处理备注
- 公布版 / 内部备注版

## 一句话用法
把学生名单、教室座位结构和排座规则给 AI，让它直接整理出一版座位表和调座说明。
