# 示例输出｜座位编排 / 调座方案

## 输出 1｜排座思路（内部说明）
```text
本次排座优先按小组合作逻辑处理：
- 1 号与 4 号优先搭配，形成带动与帮扶关系。
- 2 号与 3 号优先搭配，保证中间梯队协同推进。
- 性格冲动、纪律需关注学生不集中安排。
- 上课态度与学习态度较稳定的学生，优先分散到各组承担带动作用。
- 前排继续优先安排视力需求学生，个子高学生适当后置。
```

## 输出 2｜座位表（示意）
```text
第1行：1号A-4号A｜2号A-3号A｜1号B-4号B｜2号B-3号B
第2行：1号C-4号C｜2号C-3号C｜1号D-4号D｜2号D-3号D
……
```

## 输出 3｜学生量化摘要（内部使用）
```text
量化使用说明：
- 学业编号已按最近一次考试层次划分为 1/2/3/4。
- 纪律/专注风险为中高的学生已分散处理，未集中在同排或同组。
- 合作度、表达度、责任心较高的学生优先放在各组承担带动作用。
- 前排优先位先满足视力需求，再兼顾身高与遮挡。
```

## 输出 4｜调座说明版
```text
本次调座综合考虑了学业搭配、小组合作、课堂状态和个体差异。
请同学们先按新座位执行一周，如有特殊情况再由老师统一微调。
```
