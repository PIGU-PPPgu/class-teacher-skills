# 示例输出｜座位编排 / 调座方案

## 输出 1｜座位表（内部草稿）
```text
座位表：

第1行：李可｜王哲｜周妍｜陈雨
第2行：赵宁｜刘欣｜何睿｜林雪
第3行：张晨｜郑雅｜孙浩｜吴凡

备注：
- 李可、周妍安排在前排，优先照顾视力需求。
- 张晨放在后排，避免遮挡。
- 赵宁安排在老师视线较容易覆盖的位置。
- 赵宁与何睿未同桌。
```

## 输出 2｜调座说明版
```text
本次调座主要综合考虑了视力、身高、纪律和同桌搭配。
前排优先安排有视力需求的同学；个子较高的同学适当后置，避免遮挡；需要更多关注的同学安排在便于老师观察的位置；同时尽量保持男女搭配和整体分布均衡。

请同学们按新座位表执行。如有特殊情况，课后单独说明，不自行换位。
```
