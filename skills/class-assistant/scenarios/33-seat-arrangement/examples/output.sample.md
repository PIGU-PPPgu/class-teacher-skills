# 示例输出｜座位编排 / 调座方案

## 输出 1｜排座思路（内部说明）
```text
本次排座优先按小组合作逻辑处理：
- 1 号与 4 号优先搭配，形成带动与帮扶关系。
- 2 号与 3 号优先搭配，保证中间梯队协同推进。
- 性格冲动、纪律需关注学生不集中安排。
- 上课态度与学习态度较稳定的学生，优先分散到各组承担带动作用。
- 前排继续优先安排视力需求学生，个子高学生适当后置。
```
