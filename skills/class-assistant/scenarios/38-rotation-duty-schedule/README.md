# 场景 38｜小组轮岗 / 值周岗位轮换

适合根据小组名单、岗位设置和轮换周期，快速生成一版可执行的小组轮岗或值周岗位轮换方案。

## 这个场景解决什么问题
- 轮岗安排手工排很容易漏人、撞岗或不均衡
- 需要兼顾岗位难度、学生稳定性、轮换节奏和公平性
- 需要同时输出执行表、轮换说明和老师内部备注
- 需要避免重点岗位长期固定在少数学生身上

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：轮岗表与说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 轮岗安排表
- 值周岗位轮换表
- 执行说明
- 公布版 / 内部备注版

## 一句话用法
把小组名单、岗位清单和轮换周期给 AI，让它直接整理出一版小组轮岗或值周岗位轮换方案。
