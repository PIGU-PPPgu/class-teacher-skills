# 模板｜小组轮岗 / 值周岗位轮换

## 模板 1｜标准轮岗表
**适用：** 小组角色轮换、值周岗位安排  
**语气：** 清楚、执行导向

```text
第1周：
- 岗位A：[姓名]
- 岗位B：[姓名]
- 岗位C：[姓名]

第2周：
- 岗位A：[姓名]
- 岗位B：[姓名]
- 岗位C：[姓名]

备注：
- [备注1]
- [备注2]
```

## 模板 2｜岗位说明版
**适用：** 同步说明岗位要求与轮换逻辑  
**语气：** 简洁、可执行

```text
本轮岗位安排说明：
- 岗位A主要负责：[职责]
- 岗位B主要负责：[职责]
- 轮换逻辑：[说明]
```

## 模板 3｜学生执行版
**适用：** 班内公布或直接执行  
**语气：** 简明、正式

```text
同学们，以下为本阶段轮岗安排，请大家按时间表完成对应岗位任务。
如有特殊情况，请及时和老师沟通，不要私下自行换岗。
```
