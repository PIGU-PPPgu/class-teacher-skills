# 示例输出｜小组轮岗 / 值周岗位轮换

## 输出 1｜四周轮岗表（内部草稿）
```text
第1周：
- 组长：张晨
- 记录员：李可
- 发言人：陈雨
- 纪律提醒员：王哲

第2周：
- 组长：陈雨
- 记录员：王哲
- 发言人：李可
- 纪律提醒员：张晨
```

## 输出 2｜岗位说明版
```text
本轮岗位安排说明：
- 组长主要负责组织推进与任务收口。
- 发言人负责课堂展示与汇报。
- 轮换逻辑：先由稳定学生带一轮，再逐步加入培养对象。
```

## 输出 3｜学生执行版
```text
同学们，以下为本阶段轮岗安排，请大家按时间表完成对应岗位任务。
如有特殊情况，请及时和老师沟通，不要私下自行换岗。
```
