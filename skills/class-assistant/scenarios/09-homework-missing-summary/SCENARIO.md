# 场景 09｜作业未交 / 迟交汇总

## 场景目标
把未交、迟交、补交情况整理成一份清楚、克制、可继续跟进的汇总稿，供老师用于群提醒、私聊提醒或内部留痕。

## 适用时机
- 课后需要快速汇总谁未交、谁迟交
- 需要给家长群发统一提醒
- 需要给个别家长发定向提醒
- 需要内部留存本次作业完成情况
- 不适合直接用于公开点名羞辱学生

## 输入
- 必填信息：作业名称、截止时间、未交名单或人数
- 可选信息：迟交名单、补交截止时间、提交渠道、备注说明
- 缺失时默认：不写主观评价，只保留客观事实与下一步要求

## 输出
- 默认输出：一版班级汇总提醒
- 可选输出：家长群提醒版、私聊提醒版、内部记录版
- 输出对象：老师、家长、校内记录

## 推荐语气
- 简洁
- 明确
- 温和催办
- 克制
- 不指责

## 文件说明
- `templates.md`：未交、迟交、补交通知模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入自动统计脚本

## 直接给 AI 的说法
- 请把这份作业未交名单整理成一版适合发家长群的提醒。
- 请根据这份名单分别生成班级汇总版和一对一私聊版。
- 请把下面的未交和迟交信息整理成内部记录，不要写情绪化评价。

## 注意事项
- 默认先出草稿，不直接发送
- 不把学生贴标签，不写羞辱性表述
- 涉及长期拖欠时，建议转入一对一家长沟通场景
