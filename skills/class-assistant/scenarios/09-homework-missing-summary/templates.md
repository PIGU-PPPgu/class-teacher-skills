# 模板｜作业未交 / 迟交汇总

## 模板 1｜家长群统一提醒版
**适用：** 班级统一催交  
**语气：** 简洁、温和、明确  
**可替换字段：**
- [作业名称]
- [截止时间]
- [未交人数或名单]
- [补交截止时间]
- [提交方式]

```text
各位家长好：

今天的[作业名称]截止时间为[截止时间]。截至目前，仍有[未交人数或名单]未完成提交。
请相关同学尽快于[补交截止时间]前通过[提交方式]补交。
如有特殊情况，请及时私信说明。感谢配合。

班主任
[日期]
```

## 模板 2｜一对一提醒版
**适用：** 私聊家长提醒  
**语气：** 温和、克制  
**可替换字段：**
- [学生称呼]
- [作业名称]
- [截止时间]
- [补交截止时间]
- [备注]

```text
家长您好，想提醒您一下，[学生称呼]的[作业名称]截至[截止时间]还未提交。
麻烦您关注一下，尽量在[补交截止时间]前完成补交。
如有特殊情况，也可以及时和我说明。[备注]
```

## 模板 3｜内部记录版
**适用：** 老师留痕、后续跟进  
**语气：** 客观、简洁  
**可替换字段：**
- [作业名称]
- [截止时间]
- [未交名单]
- [迟交名单]
- [处理动作]

```text
【作业完成情况记录】
作业名称：[作业名称]
截止时间：[截止时间]
未交：[未交名单]
迟交：[迟交名单]
已采取措施：[处理动作]
后续跟进：[后续安排]
```
