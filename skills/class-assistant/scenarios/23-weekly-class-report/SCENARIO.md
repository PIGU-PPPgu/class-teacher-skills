# 场景 23｜班级周报

## 场景目标
把一周内班级的学习、纪律、活动与家校沟通情况浓缩成一份可读、可复盘、可继续跟进的周报。

## 适用时机
- 每周结束后做班级整体复盘
- 需要向自己、搭班老师或年级组汇总本周情况
- 需要把日报零散信息整合成周维度结论
- 需要沉淀下周重点工作
- 不适合替代正式总结材料或对外宣传稿

## 输入
- 必填信息：周次、班级、本周重点情况
- 可选信息：出勤趋势、学习表现、纪律问题、活动亮点、家校沟通、下周计划
- 缺失时默认：按“本周概况—亮点—问题—下周跟进”输出

## 输出
- 默认输出：一版内部周报
- 可选输出：简版汇报、家校沟通素材版、月报素材版
- 输出对象：班主任、内部协作老师

## 推荐语气
- 客观
- 概括清楚
- 不堆流水账
- 有结论感
- 便于后续行动

## 文件说明
- `templates.md`：周报结构模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入日报汇总脚本

## 直接给 AI 的说法
- 请把这周班级情况整理成一版内部周报。
- 请根据这几天日报，提炼成本周亮点、问题和下周重点。
- 请给我一版适合老师自己复盘的班级周报，不要写太空。

## 注意事项
- 周报重点是归纳，不是重复每天流水账
- 默认用于内部，不直接公开学生个体细节
- 涉及敏感情况时，保留必要结论即可
