# 场景 26｜家校沟通台账

适合把零散家校沟通记录整理成可持续维护的台账。

## 这个场景解决什么问题
- 沟通过，但后面找不到记录
- 需要区分已沟通事项与待跟进事项
- 需要做月度复盘或敏感事项留痕
- 需要给班主任自己或内部协作老师留一个统一记录面板

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、注意事项
- `templates.md`：台账表、单次沟通记录模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 家校沟通台账
- 单个学生近期沟通汇总
- 待跟进事项清单

## 一句话用法
把日期、学生、沟通对象、事由、结果、后续动作给 AI，让它整理成表格或跟进清单。
