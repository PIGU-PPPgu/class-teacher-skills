# 场景 28｜请假返校补课跟进表

适合跟进学生请假返校后的补作业、补课和确认状态。

## 这个场景解决什么问题
- 学生返校后容易出现信息断层
- 补作业、补课、补测容易漏
- 需要和家长、科任老师协同确认完成状态
- 需要给班主任留一份持续更新的返校跟进表

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、注意事项
- `templates.md`：标准返校跟进表模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 返校补课跟进表
- 待完成事项清单
- 返校衔接提醒摘要

## 一句话用法
把学生请假时间、返校时间、待补事项和确认要求给 AI，让它整理成返校跟进表。
