# 场景 31｜班级数据周统计

## 场景目标
把一周内出勤、作业、纪律、表扬、请假等班级关键数据汇总成一份简洁、可比较、便于判断趋势的周统计。

## 适用时机
- 需要用数据视角复盘一周班级运行情况
- 需要为周报、年级汇报、家长会素材做准备
- 需要从零散记录中提炼关键指标
- 需要识别本周异常点和持续问题
- 不适合替代正式统计系统报表

## 输入
- 必填信息：周次、班级、关键指标数据
- 可选信息：与上周对比、异常说明、亮点说明、下周重点
- 缺失时默认：按“指标—结果—异常—下周关注”输出

## 输出
- 默认输出：一版班级数据周统计
- 可选输出：简报版、周报素材版、趋势对比版
- 输出对象：班主任、内部协作老师、年级组

## 推荐语气
- 客观
- 概括清楚
- 趋势导向
- 简洁
- 便于复盘

## 文件说明
- `templates.md`：周统计模板与对比模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接 CSV 汇总脚本

## 直接给 AI 的说法
- 请把这一周的出勤、作业和纪律数据整理成周统计。
- 请帮我提炼本周最值得关注的异常指标。
- 请给我一版带上周对比的班级数据周报。

## 注意事项
- 优先突出趋势和异常，不要只堆数字
- 数据不完整时要标注口径或缺失项
- 如需对外展示，应再做脱敏和表达收口
