# 场景 31｜班级数据周统计

适合把一周的出勤、作业、纪律、表扬、请假等关键数据整理成趋势化周统计。

## 这个场景解决什么问题
- 零散记录很多，但没有形成一周结论
- 想知道本周和上周相比哪里变好了、哪里变差了
- 需要为周报、家长会素材、年级汇报准备数据摘要
- 需要快速抓异常点和下周重点

## 建议先看
- `SCENARIO.md`：场景说明与适用范围
- `templates.md`：标准周统计表与摘要模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 班级数据周统计表
- 一周结论摘要
- 下周关注重点

## 一句话用法
把本周关键班级数据和上周对比信息给 AI，让它整理成带趋势判断的周统计。
