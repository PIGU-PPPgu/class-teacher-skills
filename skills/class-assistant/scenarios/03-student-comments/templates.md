# 模板｜学生期末评语 / 阶段评语

## 模板 1｜标准评语版
**适用：** 期末评语、阶段总结  
**语气：** 真诚、克制、正向

```text
你在本阶段的学习和班级生活中表现出[优点]，这一点值得肯定。[若有进步点，可补：尤其在[进步点]方面，能看出你正在努力调整并取得了进步。]

同时，也希望你在今后的学习中进一步注意[需要改进的地方]，学会[具体建议]。相信只要你继续保持认真踏实的态度，今后一定会有更好的表现。
```

## 模板 2｜温暖寄语版
**适用：** 偏鼓励型评语  
**语气：** 温暖、鼓励

```text
老师看到你是一个[优点]的孩子，在[具体表现]方面给老师留下了不错的印象。如果今后你能在[改进点]方面再多下些功夫，学会[建议]，你的进步空间会更大。

期待你继续保持优点，也相信你会在不断努力中遇见更好的自己。
```

## 模板 3｜简洁批量版
**适用：** 批量快速生成  
**语气：** 简洁、规范

```text
该生在本阶段表现出[优点]，有一定的[进步点]。建议后续进一步加强[改进点]，养成[建议]的习惯。
```

## 使用提示
- 每条评语至少保留一个具体优点和一个明确建议
- 有进步点就写，没有就不虚构
- 批量写时要避免所有学生都像复制粘贴
