# 场景 03｜学生期末评语 / 阶段评语

## 场景目标
根据学生的优点、问题、进步点和建议，生成自然、克制、可交付的评语，适用于期末评语、阶段评语、成长寄语等场景。

## 适用时机
- 期末需要批量写学生评语时
- 阶段性反馈学生表现时
- 需要把零散观察整理成完整评语时

不适合：
- 使用空泛套话代替真实观察
- 写含有羞辱、标签化、定性过重的评价
- 在信息明显不足时强行写得很具体

## 输入
### 必填
- 学生优点
- 需要改进的地方

### 可选
- 进步点
- 老师期望
- 字数风格（简短 / 标准 / 温暖）
- 对象（学生本人 / 家长视角）

### 缺失时默认处理
- 未给字数：默认标准长度
- 未给对象：默认面向学生与家长都可接受的中性写法
- 未给进步点：不虚构，只保留优点与建议

## 输出
默认输出：
- 1 段可直接使用的评语

可选输出：
- 更温暖版
- 更简洁版
- 批量统一风格版

## 推荐语气
- 真诚
- 克制
- 正向
- 有具体感

## 文件说明
- `templates.md`：评语模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 帮我给小明写一段期末评语
- 这段评语太空了，帮我写得更自然一些
- 帮我统一成全班同一风格

## 注意事项
- 优点和建议都要落地，不要只堆好词
- 少用绝对化措辞
- 不写羞辱性表达
- 默认保持可公开交付的分寸
