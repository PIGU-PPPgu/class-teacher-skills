# 模板｜家访记录

## 模板 1｜标准家访记录版
**适用：** 班主任内部留档  
**语气：** 客观、完整  
**可替换字段：**
- [家访时间]
- [家访对象]
- [家访目的]
- [沟通内容]
- [家长反馈]
- [达成共识]
- [后续安排]

```text
【家访记录】
家访时间：[家访时间]
家访对象：[家访对象]
家访目的：[家访目的]

一、主要沟通内容
[沟通内容]

二、家长反馈
[家长反馈]

三、达成共识
[达成共识]

四、后续安排
[后续安排]
```

## 模板 2｜简要汇报版
**适用：** 向年级组或搭班老师同步  
**语气：** 简洁、重点明确  
**可替换字段：**
- [学生/家庭对象]
- [本次重点]
- [后续跟进]

```text
已完成对[学生/家庭对象]的家访。本次重点围绕[本次重点]进行沟通，家校已初步形成共识，后续将继续跟进[后续跟进]。
```
