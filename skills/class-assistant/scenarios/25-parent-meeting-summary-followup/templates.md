# 模板｜家长会总结与后续跟进

## 模板 1｜标准内部总结版
**适用：** 家长会后内部留档  
**语气：** 客观、清楚、可执行  
**可替换字段：**
- [时间]
- [班级]
- [会议主题]
- [会议概况]
- [核心内容]
- [家长关注点]
- [老师提醒]
- [后续安排]
- [需单独跟进]

```text
【家长会总结】
时间：[时间]
班级：[班级]
会议主题：[会议主题]

一、会议概况
[会议概况]

二、核心内容
[核心内容]

三、家长关注点
[家长关注点]

四、老师提醒
[老师提醒]

五、后续安排
[后续安排]

六、需单独跟进事项
[需单独跟进]
```
