# 示例输出｜家长会总结与后续跟进

【家长会总结】
时间：4 月 26 日晚
班级：六年级一班
会议主题：毕业阶段学习与习惯协同

一、会议概况
本次家长会围绕毕业阶段的学习安排与习惯培养展开，重点说明了近期班级整体情况及接下来需要家校共同配合的方向。

二、核心内容
重点沟通了作业落实、时间管理、复习节奏以及家校协同支持方式。

三、家长关注点
家长主要关注孩子回家后的拖延情况、电子产品使用时间以及复习效率问题。

四、老师提醒
建议家长协助学生稳定作息、减少非必要请假，并持续督促订正与作业落实。

五、后续安排
下周将继续检查作业习惯执行情况，并针对个别学生开展单独沟通。

六、需单独跟进事项
对 2 名作业拖延学生家庭进行进一步跟进，明确家校配合要求。
