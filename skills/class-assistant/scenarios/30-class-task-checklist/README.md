# 场景 30｜班级事务待办清单

适合把缴费、回执、活动准备、打印、提醒和值日调整等杂项班务统一成可执行清单。

## 这个场景解决什么问题
- 班务杂项很多，容易漏
- 任务分散在群消息、纸条、脑子里
- 需要区分待完成、待确认和已完成事项
- 需要一份班主任自己能快速扫一眼的执行面板

## 建议先看
- `SCENARIO.md`：场景目标与边界
- `templates.md`：标准待办表模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 班级事务待办表
- 今日优先处理清单
- 截止时间提醒摘要

## 一句话用法
把任务名称、截止时间、当前状态、责任对象和优先级给 AI，让它整理成班务待办清单。
