# 场景 30｜班级事务待办清单

## 场景目标
把缴费、回执、活动准备、提醒、材料收集和值日安排等班级事务整理成一份清楚、可勾选、可追踪的待办清单。

## 适用时机
- 需要汇总一周或某阶段班级杂项事务
- 需要把零散任务统一成执行清单
- 需要区分已完成、待完成、待确认事项
- 需要减少遗漏和重复提醒
- 不适合替代正式行政系统任务单

## 输入
- 必填信息：任务名称、截止时间、当前状态
- 可选信息：责任对象、提醒方式、优先级、备注
- 缺失时默认：按“任务—状态—截止时间—后续动作”输出

## 输出
- 默认输出：一版班级事务待办清单
- 可选输出：日清单、周清单、待确认事项版
- 输出对象：班主任、内部协作老师、个人执行

## 推荐语气
- 简洁
- 执行导向
- 条理清楚
- 可勾选
- 便于提醒

## 文件说明
- `templates.md`：待办清单模板与优先级模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接提醒脚本或导出脚本

## 直接给 AI 的说法
- 请把这些班务杂项整理成一份待办清单。
- 请按截止时间和优先级给我排一版班级事务列表。
- 请找出哪些事项今天必须处理，哪些可以放到本周内。

## 注意事项
- 尽量拆成可执行的小任务，不要写成空泛大项
- 区分“已完成”“待完成”“待确认”状态
- 如需外发提醒，建议从清单再生成对应提醒文案
