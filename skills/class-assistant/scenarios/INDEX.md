# 班主任场景索引

这个目录把班主任 skill 拆成一组可单独复用的小场景。

## 设计原则
- 一个场景解决一类高频任务
- 每个场景都提供：说明、模板、输入输出样例
- 默认先生成草稿，不直接外发
- 默认不处理未经核实的敏感信息
