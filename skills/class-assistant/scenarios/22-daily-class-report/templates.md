# 模板｜班级日报

## 模板 1｜标准内部日报版
**适用：** 班主任内部留档、交接  
**语气：** 客观、简洁  
**可替换字段：**
- [日期]
- [班级]
- [出勤情况]
- [学习情况]
- [纪律情况]
- [重点事件]
- [家校沟通]
- [后续跟进]

```text
【班级日报】
日期：[日期]
班级：[班级]

一、出勤情况
[出勤情况]

二、学习情况
[学习情况]

三、纪律与常规
[纪律情况]

四、重点事件
[重点事件]

五、家校沟通
[家校沟通]

六、后续跟进
[后续跟进]
```

## 模板 2｜精简交接版
**适用：** 搭班老师快速查看  
**语气：** 简明、重点突出  
**可替换字段：**
- [今日重点]
- [需关注学生]
- [明日提醒]

```text
今日重点：[今日重点]
需关注：[需关注学生]
明日提醒：[明日提醒]
```
