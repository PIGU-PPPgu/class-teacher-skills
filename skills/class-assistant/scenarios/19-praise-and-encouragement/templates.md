# 模板｜表扬通报 / 班级激励

## 模板 1｜班级公开表扬版
**适用：** 班内或群内正向通报  
**语气：** 真诚、简洁  
**可替换字段：**
- [表扬对象]
- [具体表现]
- [发生场景]
- [鼓励方向]

```text
今天想表扬[表扬对象]。在[发生场景]中，[他/她]表现出了[具体表现]。
这样的表现值得肯定，也希望大家一起向好的行为看齐，继续保持[鼓励方向]。
```

## 模板 2｜家长私聊反馈版
**适用：** 向家长反馈学生正向表现  
**语气：** 温和、自然  
**可替换字段：**
- [学生姓名]
- [进步表现]
- [老师观察]

```text
家长您好，想和您分享一下[学生姓名]最近的一个积极变化：
在[老师观察]中，可以看到孩子在[进步表现]方面有了明显进步。希望我们继续一起鼓励孩子，把这种状态保持下去。
```
