# 模板｜值日 / 轮值安排

## 模板 1｜按周轮值安排版
**适用：** 周值日、岗位轮值  
**语气：** 清楚、执行导向  
**可替换字段：**
- [岗位]
- [周期]
- [人员名单]
- [值日要求]

```text
岗位名称：[岗位]
轮值周期：[周期]

第 1 周：[学生 1]、[学生 2]
第 2 周：[学生 3]、[学生 4]
第 3 周：[学生 5]、[学生 6]
第 4 周：[学生 7]、[学生 8]

值日要求：
- [要求 1]
- [要求 2]
- [要求 3]

如遇请假或特殊情况，请提前和老师说明，再做调换。
```

## 模板 2｜按日值日安排版
**适用：** 每日值日、短周期轮换  
**语气：** 简洁、明确  
**可替换字段：**
- [日期/星期]
- [岗位]
- [人员]

```text
周一：[岗位] - [人员]
周二：[岗位] - [人员]
周三：[岗位] - [人员]
周四：[岗位] - [人员]
周五：[岗位] - [人员]
```

## 模板 3｜班级公布说明版
**适用：** 发班级群或班级公告  
**语气：** 正式、简洁  
**可替换字段：**
- [安排周期]
- [执行要求]

```text
同学们，以下是本阶段班级值日/轮值安排，请大家按要求认真完成。
如有特殊情况需要调整，请提前说明，不要临时缺岗。

安排周期：[安排周期]
执行要求：[执行要求]
```
