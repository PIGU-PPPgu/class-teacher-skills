# 示例输出｜积分量化管理系统

## 输出 1｜积分规则表（学生公布版）
```text
七年级某班积分量化管理规则

一、考勤常规
- 一周无迟到缺勤 +2分
- 每迟到超过5分钟 -5分
- 正式上课未落座 -1分
- 未佩戴校卡 -5分
- 校服不合格 -10分

二、纪律
- 讲粗口脏话 -5分
- 下课追逐打闹 -5分
- 破坏公物 -10分
- 殴打同学 -20分

三、卫生
- 高效完成值日 +2分
- 未完成值日 -5分
- 导致班级被扣分 -10分
- 一周个人卫生整洁 +1分
- 周围地面有垃圾 -2分

四、学习
- 主动举手回答问题 +1分
- 课堂展示 +1分
- 周测单科第一 +3分
- 期中/期末单科状元 +10分
- 作业A+ +5分、A +2分
- 未按时交作业 -10分
- 抄袭作业：当周积分全部清零
- 考试作弊 -50分

五、体育跑操
- 跑操迟到/未坚持/打闹/无声 各-2分

积分每周结算，由学习委员周一公示。
```

## 输出 2｜游戏化系统核心规则
```text
班级大富翁规则摘要

1. 每人初始500班级货币
2. 蓝色地产可购买，租金100元/周，4人共租则每人25元
3. 职位薪资：银行行长200/周、统计局长200/周、卫生局长150/周
4. 抽卡：小组200元/次，个人50元/次
5. 银行存款500元起，周利息10%
6. 面壁思过区罚款200元
7. 学期继承：1000→200、2000→500、3000+→1000
8. 积分最高者可优先选座位

财富榜每周公布，上榜者免费抽牌一次。
```

## 输出 3｜收集流程与角色权限
```text
积分收集流程

打分权限：
- 班主任：所有维度，直接打分
- 科任老师：课堂表现、作业，可委托课代表
- 课代表：本学科作业质量（A+/A/未交/不合格）、课堂表现
- 值日班长：当日考勤、纪律、卫生、自习（每日轮换）
- 学习委员：汇总收集、公示、纠错调整

每日流程：
1. 课代表根据当天情况在行为簿打分
2. 值日班长记录当日综合情况，放学前交学习委员
3. 学习委员每周一汇总公示

公示内容：
- 个人积分前十名
- 小组积分排名
- 本周加减分说明
```
