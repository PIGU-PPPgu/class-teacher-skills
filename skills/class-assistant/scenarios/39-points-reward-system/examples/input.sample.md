# 示例输入｜积分量化管理系统

## 示例 1｜建立积分规则
```text
班级：七年级某班
学生人数：45人
分组：9个小组，每组5人

需求：
- 设计一套覆盖考勤、纪律、学习、卫生、体育、常规的积分规则
- 扣分不要太重，正向激励为主
- 需要让课代表参与日常打分
- 希望引入游戏化机制，用"班级货币"概念

已有工具：
- 学生行为簿小程序（行为打分）
- 班级小管家小程序（成绩发布、信息收集）

特殊要求：
- 抄袭作业属于严重违规，需重罚
- 考试作弊直接扣50分
- 每周公布积分排名
```

## 示例 2｜设计游戏化系统
```text
班级：七年级某班
已有一套基础加减分规则

希望增加：
- 班级大富翁玩法，学生可以用积分参与
- 有地产、租金、职位薪资的概念
- 有抽卡机制增加趣味性
- 学期末积分可以部分继承到下学期
- 积分高的小组和个人有实际奖励（如选座位）

参考设定：
- 初始货币500元
- 银行利息每周10%
- 职位有银行行长、统计局长等，每周有薪资
```

## 示例 3｜整理收集流程
```text
班级：七年级某班
已有积分规则，需要明确谁负责打分、怎么汇总

角色：
- 班主任
- 科任老师（5位）
- 各科课代表（9位）
- 值日班长（每日轮换，共9人）
- 学习委员（1人）

工具：
- 学生行为簿小程序

希望输出：
- 各角色打分权限表
- 每日/每周操作流程
- 公示时间节点
```
