## 模板 2｜积分收集与汇总流程
**适用：** 班级内部打分操作流程  
**语气：** 简明、流程化

```text
[班级]积分收集流程

一、打分角色与权限

| 角色           | 打分权限                     | 打分方式             |
|----------------|------------------------------|----------------------|
| 班主任         | 所有维度                     | 直接打分             |
| 科任老师       | 课堂表现、作业               | 直接打分或委托课代表 |
| 课代表         | 本学科作业质量、课堂表现     | 在行为簿小程序打分   |
| 值日班长       | 当日考勤、纪律、卫生、自习   | 在行为簿小程序打分   |
| 学习委员       | 收集汇总、公示、纠错         | 汇总后直接加减调整   |

二、每日流程

1. 课代表：根据当天作业质量和课堂表现，在[行为簿平台]为对应学生打分
2. 值日班长（每日轮换）：
   - 记录当日考勤、纪律、卫生情况
   - 当天放学前算清每人当日总分
   - 交学习委员汇总
3. 学习委员：
   - 收集各渠道积分数据
   - 每周[固定时间]公示积分排名
   - 对错误数据可直接加减调整

三、推荐工具

- 行为打分：学生行为簿小程序（预设评价模板和指标）
- 成绩发布/信息收集：班级小管家小程序
- 也可使用纸质表格或在线表格替代
```

## 模板 3｜游戏化积分系统（班级大富翁）
**适用：** 希望通过游戏化机制提升学生积极性的班级  
**语气：** 有趣、激励、规则清晰

```text
[班级]大富翁积分游戏规则

一、基本设定
- 每位同学初始获得[500]班级货币
- 积分可通过日常加减分、职位工资、抽卡奖励等方式获取
- 积分也可通过购买地产、抽卡、违规罚款等方式消耗

二、地产与租金
- 蓝色地产：可供购买，购买后每周收取租金[100]元
- 小组成员共同使用时，租金由[4]人平均承担（每人[25]元/周）
- 绿色/黄色区域为政府用地，红色为公摊走廊，不可购买

三、班级职位与薪资
| 职位           | 周薪（班级货币） |
|----------------|------------------|
| 银行行长       | 200              |
| 统计局长       | 200              |
| 卫生管理局长   | 150              |
| 司法局长       | 100              |
| 纪检委会长     | 50/天            |

四、抽卡规则
- 小组抽卡：[200]元/次
- 个人抽卡：[50]元/次
- 牌型：A-K + 大小王 + 金卷
- 财富榜上榜者可获得免费抽牌机会

五、惩罚区
- 面壁思过区：需支付[200]班级货币
- 严重违规：视情况扣除积分或没收资产

六、银行系统
- 最低存款：[500]元
- 每周利息：[10%]

七、学期结算与继承
| 上学期期末余额 | 新学期初始余额 |
|----------------|----------------|
| 1000元         | 200元          |
| 2000元         | 500元          |
| 3000元及以上   | 1000元         |

八、积分兑换奖励
- 个人积分排名第一：可优先选择座位
- 小组积分排名第一：可优先选择小组位置
- 其他奖励由班主任根据实际情况设定
```

## 模板 5｜课代表打分授权说明
**适用：** 向课代表说明打分权限和操作方式  
**语气：** 简明、可操作

```text
课代表打分授权说明

一、你的权限
- 你可以为全班同学在本学科维度内加减分
- 加分：作业A+（+5）、A（+2）、课堂表现优秀（+1）
- 扣分：未交作业（-10）、作业不合格（-5）

二、操作方式
- 打开[行为簿小程序]
- 选择对应学生 → 选择加分/扣分项目 → 确认
- 如不确定分值，可先记录后请班主任确认

三、注意事项
- 打分要公平，不偏袒、不报复
- 每天放学前完成当天打分
- 遇到争议情况及时汇报班主任
```
