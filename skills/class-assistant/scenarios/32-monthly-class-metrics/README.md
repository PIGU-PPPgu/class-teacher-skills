# 场景 32｜班级数据月统计

适合把一个月的班级关键数据整理成带趋势、异常点和下月动作的月统计。

## 这个场景解决什么问题
- 月度复盘不能只靠感觉，需要数据支撑
- 需要为月报、家长会、阶段总结准备数据材料
- 需要识别持续异常与长期改善趋势
- 需要把多周记录提炼成管理判断

## 建议先看
- `SCENARIO.md`：场景目标、输出方式、注意事项
- `templates.md`：标准月统计表与结论模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 班级数据月统计表
- 月度结论摘要
- 下月重点动作

## 一句话用法
把月度关键班级数据、周度趋势和异常点给 AI，让它整理成可复盘的月统计。
