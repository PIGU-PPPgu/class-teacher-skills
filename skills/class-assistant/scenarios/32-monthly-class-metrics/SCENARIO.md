# 场景 32｜班级数据月统计

## 场景目标
把一个月内班级的出勤、作业、纪律、表扬、请假和重点事项沉淀成一份有趋势感、可复盘、可支持管理判断的月统计。

## 适用时机
- 需要月度维度看班级整体运行趋势
- 需要为月报、家长会、阶段总结准备数据材料
- 需要从周统计和日常记录中提炼月度结论
- 需要识别长期异常和持续改善点
- 不适合替代正式校务统计报表

## 输入
- 必填信息：月份、班级、关键指标数据
- 可选信息：周度趋势、重点异常、亮点、改进建议、下月重点
- 缺失时默认：按“指标—趋势—关注点—下月动作”输出

## 输出
- 默认输出：一版班级数据月统计
- 可选输出：阶段汇报版、家长会素材版、管理复盘版
- 输出对象：班主任、内部协作老师、年级组

## 推荐语气
- 客观
- 趋势导向
- 有结论感
- 简洁
- 便于规划

## 文件说明
- `templates.md`：月统计模板与趋势摘要模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接周统计汇总脚本

## 直接给 AI 的说法
- 请把这个月的班级数据整理成月统计。
- 请提炼本月最明显的趋势和需要持续跟进的问题。
- 请给我一版适合做月报素材的班级数据总结。

## 注意事项
- 月统计重点在趋势和结论，不是简单罗列周数据
- 数据口径不一致时要标注
- 如用于公开展示，应再次脱敏和收口表达
