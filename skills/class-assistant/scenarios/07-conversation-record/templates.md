# 模板｜谈话记录

## 模板 1｜标准结构化记录版
**适用：** 日常谈话、问题沟通、教育引导  
**语气：** 客观、清晰  
**可替换字段：**
- [时间]
- [地点]
- [谈话对象]
- [参与人员]
- [背景]
- [内容摘要]
- [学生状态]
- [共识/约定]
- [跟进计划]

```markdown
### 谈话记录

**基本信息**
- 时间：[时间]
- 地点：[地点]
- 谈话对象：[谈话对象]
- 参与人员：[参与人员]

**背景**
[背景]

**谈话内容摘要**
1. [内容要点 1]
2. [内容要点 2]
3. [内容要点 3]

**学生状态观察**
- 情绪状态：[学生状态]
- 配合程度：
- 认知表现：

**达成的共识 / 约定**
- [共识/约定 1]
- [共识/约定 2]

**后续跟进计划**
- [跟进计划 1]
- [跟进计划 2]
- 预计回访时间：
```

## 模板 2｜简版存档摘要
**适用：** 快速记录、后续补充正式版  
**语气：** 简洁、留痕

```markdown
### 谈话摘要
- 时间：[时间]
- 对象：[谈话对象]
- 事由：[背景]
- 结论：[本次主要结论]
- 后续跟进：[下一步动作]
```

## 使用提示
- 优先记录事实与原话要点，不做过度心理推断
- 先写谈话背景，再写内容，再写跟进
- 记录的目的是支持后续教育，不是为了堆材料
