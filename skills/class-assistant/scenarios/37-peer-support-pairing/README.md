# 场景 37｜学生互助结对 / 帮扶对象分配

适合根据学生名单、学习层次和班级管理要求，快速生成一版可执行的互助结对或帮扶分配方案。

## 这个场景解决什么问题
- 互助结对如果只看成绩，容易忽略性格、责任心和稳定性
- 需要兼顾带动关系、帮扶目标、沟通方式和持续跟进
- 需要避免不合适的搭配或让重点学生过度集中
- 分配后还需要一版老师内部说明与学生执行版

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：结对表与说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 互助结对表
- 帮扶目标建议版
- 跟进说明
- 公布版 / 内部备注版

## 一句话用法
把学生名单、结对目标和搭配规则给 AI，让它直接整理出一版学生互助结对或帮扶对象分配方案。
