# 示例输入｜学生互助结对 / 帮扶对象分配

## 示例 1｜学习互助结对
```text
班级人数：12 人
结对目标：建立一帮一互助关系，兼顾学习带动与课堂提醒

学生名单：
1. 张晨（学习较强，责任心强）
2. 李可（学习中上，表达较好）
3. 王哲（学习中，状态稳定）
4. 陈雨（学习较强，做事认真）
5. 赵宁（学习波动，课堂提醒需求较多）
6. 刘欣（学习中，执行较稳）
7. 孙浩（学习较强，行动力强）
8. 周妍（学习中上，沟通较好）
9. 吴凡（学习中，容易分心）
10. 郑雅（学习中，认真细致）
11. 何睿（学习中下，性格内向）
12. 林雪（学习较强，组织能力较好）

额外规则：
- 优先考虑高低搭配与稳定带动
- 不让课堂状态都偏弱的学生配在一起
- 性格过于冲突的学生不要互相结对
- 需要同时给出一版老师内部跟进说明
```
