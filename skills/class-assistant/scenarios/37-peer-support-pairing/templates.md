# 模板｜学生互助结对 / 帮扶对象分配

## 模板 1｜标准结对表
**适用：** 学习互助、一帮一帮扶初稿  
**语气：** 清楚、执行导向

```text
学生互助结对表：

第1对：[姓名A] ↔ [姓名B]
第2对：[姓名A] ↔ [姓名B]
第3对：[姓名A] ↔ [姓名B]

备注：
- [备注1]
- [备注2]
```

## 模板 2｜帮扶目标建议版
**适用：** 需要同步给出每对的帮扶方向  
**语气：** 简洁、可执行

```text
第1对：[姓名A] ↔ [姓名B]
- 本阶段目标：[目标]
- 跟进方式：[方式]

第2对：[姓名A] ↔ [姓名B]
- 本阶段目标：[目标]
- 跟进方式：[方式]
```

## 模板 3｜学生执行版
**适用：** 班内公布或对学生说明  
**语气：** 简明、正式

```text
同学们，以下为本阶段学生互助结对安排，请大家按要求开展学习互助与日常提醒。
如有特殊情况，请及时和老师沟通，不要私下自行更换搭档。
```
