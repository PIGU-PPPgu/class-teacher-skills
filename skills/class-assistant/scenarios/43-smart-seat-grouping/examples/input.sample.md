# 输入样例｜48 人班级多维评分表（脱敏）

以下为一个 48 人班级的学生多维评分表片段，以 CSV 格式呈现。
所有姓名均为占位符，不代表任何真实学生。

## 评分说明

- 学业排名段：1=前12名, 2=13-24名, 3=25-36名, 4=37名及以后
- 偏科程度：0=均衡, 1=严重偏科
- 外向话多：0=安静内向, 1=极度外向话多
- 自控力：0=自控力弱, 1=自控力强
- 领导力：0=被动跟随, 1=强领导力
- 情绪稳定性：0=波动大, 1=非常稳定
