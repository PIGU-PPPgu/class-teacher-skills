# 模板｜智能座位编排系统

## 模板 1｜学生多维评分表结构
**适用：** 班主任对学生进行多维评分，作为编排输入  
**语气：** 客观、简洁  
**字段说明：**

| 字段 | 说明 | 取值范围 |
|------|------|----------|
| 姓名 | 学生姓名 | 文本 |
| 性别 | 男/女 | M/F |
| 学业排名段 | 按成绩排名分段 | 1=前12名, 2=13-24名, 3=25-36名, 4=37名及以后 |
| 偏科程度 | 学科发展不均衡程度 | 0-1（0=均衡, 1=严重偏科） |
| 外向话多 | 外向程度与话多倾向 | 0-1（0=安静内向, 1=极度外向话多） |
| 自控力 | 课堂自控能力 | 0-1（0=自控力弱, 1=自控力强） |
| 领导力 | 组织协调与主动性 | 0-1（0=被动跟随, 1=强领导力） |
| 情绪稳定性 | 情绪波动与抗压能力 | 0-1（0=波动大, 1=非常稳定） |
| 备注 | 特殊情况说明 | 文本，可选 |

**示例行（CSV 格式）：**

```csv
姓名,性别,学业排名段,偏科程度,外向话多,自控力,领导力,情绪稳定性,备注
张三,M,1,0.3,0.7,0.6,0.8,0.7,数学课代表
李四,F,2,0.6,0.4,0.8,0.5,0.9,偏文科
王五,M,3,0.4,0.8,0.3,0.6,0.5,
赵六,F,4,0.7,0.3,0.5,0.2,0.6,需要关注
```

**使用提示：**
- 评分可基于期中/期末成绩和日常观察综合打分
- 不必过度精确，0.1 的差异不会显著影响编排结果
- 备注栏可标注特殊约束（如视力、身高、不可同组等）

## 模板 2｜分组结果公示版
**适用：** 向学生或家长公布分组结果  
**语气：** 正式、公平、正向

```text
[班级]智能分组方案公示

各位同学/家长：

经过综合评估，[班级]已完成新学期学习小组编排。
本次编排综合考虑了学业水平、性格特点、自控能力等多方面因素，
旨在实现组内优势互补、组间实力均衡。

分组结果如下：

第1组：[姓名A]、[姓名B]、[姓名C]、[姓名D]
第2组：[姓名E]、[姓名F]、[姓名G]、[姓名H]
...
第12组：[姓名XX]、[姓名YY]、[姓名ZZ]、[姓名WW]

说明：
- 分组综合考虑学业层次与性格互补
- 每组包含不同学业层次的同学，便于互助学习
- 如有特殊困难或合理调整需求，请于[日期]前联系班主任
- 本方案自[日期]起执行

班主任
[日期]
```

## 模板 3｜调座说明模板
**适用：** 向学生和家长说明调座原因与规则  
**语气：** 客观、平和、正向引导

```text
[班级]调座说明

一、调座原因
为促进同学间的互助学习和良性互动，班级将于[日期]进行座位调整。

二、编排原则
1. 学业互补：成绩层次不同的同学搭配，便于互助
2. 性格对冲：外向与内向搭配，活跃与沉稳搭配
3. 偏科互补：不同学科优势的同学组合
4. 情绪平衡：情绪稳定性不同的同学互相影响

三、注意事项
- 调座后有两周适应期，期间不接受换组申请
- 如有视力、身高等方面特殊需求，请提前联系班主任
- 分组结果仅作为座位和小组活动依据，不代表任何评价

四、执行时间
- 公布日期：[日期]
- 执行日期：[日期]
- 适应期：[日期]至[日期]

班主任
[日期]
```

## 模板 4｜家长说明信
**适用：** 向家长解释智能分组编排的目的和原则  
**语气：** 温和、专业、正向

```text
尊敬的家长：

您好！

新学期伊始，[班级]将进行学习小组和座位的重新编排。本次编排
采用"智能分组"方式，基于学生的学业表现、学习特点和性格特征
进行综合匹配，力求实现以下目标：

1. 优势互补：让不同学科优势的同学互相帮助
2. 性格平衡：让外向和内向的同学互相影响、共同成长
3. 公平均衡：各组整体实力接近，营造良性竞争氛围

分组方案已经过认真评估，但每位同学的独特性无法完全用数据衡量。
如果您的孩子在新的小组中有任何不适应，欢迎及时与我沟通。

感谢您的理解与支持！

班主任 [姓名]
[日期]
```
