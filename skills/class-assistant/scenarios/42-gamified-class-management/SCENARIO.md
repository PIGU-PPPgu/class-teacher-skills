# 场景 42｜游戏化班级管理 / 大富翁式班级管理系统

## 场景目标
提供一套完整的游戏化班级管理方案，将大富翁机制融入日常班级管理，覆盖货币系统、职位系统、地产系统、卡片系统和排行榜，帮助班主任用游戏化思维激发学生内驱力。

## 适用时机
- 新学期建立班级管理制度
- 常规纪律/学习/卫生激励效果不足，需要引入趣味机制
- 希望培养学生的经济意识、责任意识和规则意识
- 已有基础班规，需要增加激励层
- 不适合替代学校正式的纪律处分制度
- 不适合缺乏执行力的班级（需先建立基本秩序）

## 输入
- 必填信息：班级人数、年级、已有班规（如有）
- 可选信息：希望重点管理的维度（考勤/纪律/卫生/学习/作业）、是否分组、班级现有积分制度
- 缺失时默认：按标准50人班级输出完整游戏化管理方案

## 输出
- 默认输出：完整的游戏化班级管理规则手册（含货币、职位、地产、卡片、排行榜全系统）
- 可选输出：单项子系统说明、学生版规则手册、家长说明信、积分记录表
- 输出对象：班主任（内部管理版）、学生（执行版）、家长（知情版）

## 推荐语气
- 趣味生动
- 规则清晰
- 公平公正
- 正向激励为主

## 文件说明
- `templates.md`：各子系统模板与规则模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 我想在班级里引入大富翁游戏化管理，帮我设计一整套规则。
- 我班40人，想重点管纪律和作业，请帮我定制一套简化版游戏化管理方案。
- 帮我写一份给家长的说明信，解释班级大富翁制度。
- 我想调整卡片系统的概率，帮我重新设计牌面效果。

## 注意事项
- 游戏化管理是辅助手段，不能替代正常的师生关系和情感教育
- 负面积分（惩罚）设置应克制，避免过度惩罚导致学生抵触
- 定期根据班级实际情况调整规则，保持游戏新鲜感
- 所有示例均为脱敏处理，不包含真实学生信息
