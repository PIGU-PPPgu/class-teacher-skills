# 示例输出｜游戏化班级管理

## 输出 1｜完整方案框架（节选）
```text
=== 七年级某班大富翁班级管理系统 ===

一、货币系统
  货币名称：星光币
  面值：1、5、10、50、100
  获取：全勤+10/周，作业全对+5/次，课堂积极+2/次
  银行存储：500起存，10%/周利息
  继承规则：转学/毕业可指定继承

二、职位系统（8个职位）
  银行行长（100/周）：管理货币
  纪检委主任（80/周）：监督纪律
  司法局长（80/周）：处理纠纷
  卫生局长（70/周）：管理卫生
  统计局长（70/周）：记录积分
  教育局长（70/周）：督促学习
  文化旅游局长（60/周）：组织活动
  监狱长（60/周）：管理惩罚执行

三、地产系统
  蓝色（可购买）：VIP座位、景观座位等
  绿色/黄色（政府/公共）：图书角等
  红色（公摊）：走廊等
  租金递增：4周后涨至15%，8周后20%

四、卡片系统
  A-K + 大小王 + 金卷
  正向卡：选座权、免作业、双倍积分等
  负向卡：罚款、入狱等
  特殊卡：免死金牌、时光倒流等

五、排行榜
  牛布斯榜（正分前10，公开）
  跨布斯榜（负分前5，仅班主任可见）

六、奖励兑换
  100=免费抽牌，200=选座位，500=免作业
  1000=班级特权卡

七、加减分标准
  考勤/纪律/卫生/学习/作业各有明确分值
  收集流程：值日班长→学习委员→班主任审核→统计局公示
```

## 输出 2｜简化版方案（节选）
```text
=== 五年级某班星光币系统（简化版） ===

一、如何赚星光币
  好事            星光币
  全天没被点名    +3
  作业按时交      +2
  课堂积极举手    +1/次
  帮助同学        +2
  考试进步        +5~+15

二、如何花星光币
  星光币       可以换什么
  30          抽一张奖励卡
  50          自选座位一天
  100         免一次作业
  200         当一天"小老师"

三、奖励卡（6种）
  免死金牌：抵消一次小错误
  幸运卡：随机获得5-30星光币
  换座卡：自选座位一天
  免作业卡：免一次作业
  惊喜卡：班主任送一个小礼物
  组队卡：指定一位同学一起赚双倍

四、每周更新
  每周五公布本周星光币排行
  前三名获得"本周之星"贴纸

规则简单版就这些，运行一个月后再根据情况调整。
```
