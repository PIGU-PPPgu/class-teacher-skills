# 模板｜游戏化班级管理 / 大富翁式班级管理系统

## 模板 1｜货币系统
**适用：** 班级经济体系基础  
**语气：** 清晰、可执行  

```text
=== 班级货币系统 ===

货币名称：[班级币名]（如：星光币、班级金币）
发行机构：班级银行
基础面值：1、5、10、50、100

获取途径：
  日常行为：
  - 全勤一周：+10
  - 课堂积极回答：+2/次
  - 作业全对：+5/次
  - 考试进步：+10-30（按幅度）
  - 卫生值日优秀：+5
  - 帮助同学：+3
  - 班级贡献（如主动承担任务）：+5-20

  职位工资：
  - 各岗位按职责发放周薪（见职位系统）

支出途径：
  - 购买地产（见地产系统）
  - 缴纳罚款（违纪扣分对应金额）
  - 抽卡费用
  - 拍卖/特殊活动参与费

银行存储规则：
  - 起存金额：500
  - 存款利率：10%/周（鼓励储蓄）
  - 取款需提前申请
  - 可继承：学生转学/毕业时余额可指定继承给班级任意同学
```

## 模板 2｜职位系统
**适用：** 班级管理岗位设置  
**语气：** 清晰、职责明确  

```text
=== 班级职位系统 ===

职位一：银行行长
  工资：100/周
  职责：管理班级货币发行与回收、记录存款与利息、审核取款申请
  要求：数学好、诚信

职位二：纪检委主任
  工资：80/周
  职责：监督课堂纪律、记录违纪情况、执行扣分判定
  要求：公正、有威信

职位三：司法局长
  工资：80/周
  职责：处理同学间纠纷、判定规则争议、组织"听证会"
  要求：沟通能力强、讲道理

职位四：卫生管理局局长
  工资：70/周
  职责：安排和检查卫生值日、评定卫生等级、管理公共区域
  要求：责任心强

职位五：统计局长
  工资：70/周
  职责：统计每日积分变化、更新排行榜、公示数据
  要求：细心、数据敏感

职位六：教育局局长
  工资：70/周
  职责：督促作业完成、统计考试成绩变化、组织学习互助
  要求：学习成绩优秀

职位七：文化旅游局局长
  工资：60/周
  职责：策划班级文化活动、管理班级图书角、组织课间活动
  要求：有创意、活跃

职位八：监狱长
  工资：60/周
  职责：管理"入狱"同学（严重违纪者）、监督反思任务执行
  要求：公正严格

职位竞选规则：
  - 每学期初公开竞选
  - 任期一学期，期中可申请调岗
  - 失职者可被弹劾（需全班过半同意）
```

## 模板 3｜地产系统
**适用：** 班级空间经营权拍卖  
**语气：** 趣味、策略性  

```text
=== 班级地产系统 ===

地块分类：

蓝色地块（可购买）：
  - 讲台旁VIP座
  - 窗边景观座
  - 空调最佳位
  起拍价：200-500
  租金：持有者每周收取路过者 [地标价×10%]

绿色地块（政府管理）：
  - 图书角
  - 储物柜区
  使用需向文化旅游局申请，缴纳使用费

黄色地块（公共福利）：
  - 饮水区
  - 洗手间通道
  免费使用，但需遵守公共秩序

红色地块（公摊区域）：
  - 走廊
  - 黑板区
  所有人共同维护，破坏者罚款

租金递增机制：
  - 基础租金按地标价10%/周
  - 连续持有满4周：租金涨至15%/周
  - 连续持有满8周：租金涨至20%/周
  - 转让时租金重置

拍卖规则：
  - 每月初公开拍卖空置地块
  - 起拍价由班主任设定
  - 最高价得标
```

## 模板 4｜卡片系统
**适用：** 随机事件与惊喜机制  
**语气：** 趣味、不可预测  

```text
=== 班级卡片系统 ===

卡片类型：
  A-K牌型（13种）+ 大小王 + 金卷

牌面效果示例：

正向卡：
  A - 选座权：本周可自由选择座位一次（+100）
  2 - 免作业权：可免除一次作业（+50）
  3 - 双倍积分：本周所有收入翻倍（+80）
  4 - 好运降临：随机获得50-200奖励（+50-200）
  5 - 免费抽牌：可再抽一张牌（+30）

中性卡：
  6 - 位置互换：与任意同学互换座位一天（0）
  7 - 任务卡：完成指定任务获得奖励（完成任务+50，未完成-30）
  8 - 合作卡：指定一位同学与你共享本周收入（0）

负向卡：
  9 - 罚款通知：缴纳50罚款（-50）
  10 - 入狱通知：课间禁止外出一次（-30）
  J - 税务稽查：缴纳存款的10%（-存款×10%）
  Q - 地震：名下所有地产租金本周减半（-租金收入）

特殊卡：
  K - 涨薪令：本周职位工资翻倍（+工资）
  小王 - 免死金牌：抵消一次惩罚（稀有）
  大王 - 时光倒流：撤销本周一次扣分（稀有）
  金卷 - 任意门：可指定获得任意一张正向卡效果（超稀有）

抽卡规则：
  - 每周表现优秀者获得1次免费抽卡机会
  - 额外抽卡需支付50/次
  - 每人每周最多抽3次
```

## 模板 5｜排行榜与奖励
**适用：** 正负积分排行与兑换  
**语气：** 激励、透明  

```text
=== 排行榜系统 ===

正分榜（牛布斯榜）：
  - 展示前10名积分最高的同学
  - 每周一更新，由统计局负责
  - 榜首获得"周冠军"称号

负分榜（跨布斯榜）：
  - 展示积分最低的5名同学（仅班主任可见，不公开张贴）
  - 用于班主任内部关注和个别辅导

奖励兑换机制：
  积分        可兑换奖励
  100        免费抽牌1次
  200        选座位1次
  300        选同桌1次
  500        免作业1次
  800        指定座位1周
  1000       班级特权卡（自定一项合理特权）
  1500       与班主任共进午餐

奖励发放规则：
  - 每月末集中兑换
  - 兑换后扣除对应积分
  - 特权使用需提前一天申请
  - 不可转让、不可兑换现金
```

## 模板 6｜加减分标准
**适用：** 各维度行为评分  
**语气：** 明确、公平  

```text
=== 行为加减分标准 ===

考勤维度：
  全勤一周        +10
  迟到1次         -5
  早退1次         -5
  无故旷课        -20
  请假（合规）     0

纪律维度：
  课堂被表扬      +3
  课堂被点名批评  -5
  课间追逐打闹    -5
  损坏公物        -10~−30（按程度）
  打架            -30
  考试作弊        -50

卫生维度：
  值日优秀        +5
  值日不合格      -5
  主动维护卫生    +3
  乱扔垃圾        -5
  个人区域脏乱    -3

学习维度：
  作业全对        +5
  作业未交        -10
  作业迟交        -5
  考试进步明显    +10~+30
  考试退步明显    -5~-15
  课堂积极发言    +2/次

积分收集流程：
  1. 值日班长每日记录全班加减分
  2. 学习委员每日汇总并交班主任审核
  3. 统计局每周一公示上周积分变动
  4. 如有异议，向司法局长申诉
  5. 班主任有最终裁定权
```

## 模板 7｜家长说明信
**适用：** 向家长介绍游戏化管理制度  
**语气：** 温和、说明性  

```text
各位家长：

您好！为了更好地激发孩子们的学习积极性和自我管理能力，本学期班级将试行一套"大富翁"式游戏化班级管理方案。

简单来说，就是把班级日常管理（出勤、作业、纪律、卫生等）变成一个有趣的游戏系统：

- 孩子们通过良好表现赚取"班级币"
- 可以竞选"班级职位"（如银行行长、纪检委主任等）
- 可以"购买"座位使用权或抽取奖励卡
- 有排行榜展示正向表现

这个制度的目的是：
1. 用正向激励代替简单惩罚
2. 培养孩子的经济意识和规则意识
3. 让班级管理更有趣，增强参与感

请注意：
- 这套系统不涉及任何真实金钱
- 所有奖惩都在班级内部进行
- 我们会根据实际运行情况持续优化

如果您有任何疑问或建议，欢迎随时和我沟通。

班主任
[日期]
```
