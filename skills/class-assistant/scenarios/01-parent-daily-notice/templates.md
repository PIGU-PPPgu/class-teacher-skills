# 模板｜家长群日常通知

## 模板 1｜标准正式版
**适用：** 常规家长群通知  
**语气：** 正式、简洁  
**可替换字段：**
- [事项]
- [时间]
- [地点]
- [要求]
- [补充说明]

```text
各位家长好：

明天[事项]，请孩子于[时间]前到达[地点]，并提前准备好[要求]。
[补充说明]

感谢配合！
[班主任姓名]
[日期]
```

## 模板 2｜简洁提醒版
**适用：** 群内快速提醒  
**语气：** 简短、明确

```text
家长您好，提醒一下：明天[事项]，请孩子于[时间]前到达[地点]，并准备好[要求]。感谢配合。
```

## 模板 3｜需要回执确认版
**适用：** 需确认收到 / 需反馈的通知  
**语气：** 明确、可执行

```text
各位家长好：

关于[事项]，请于[截止时间]前回复“收到”，并协助孩子准备好[要求]。
时间：[时间]
地点：[地点]

感谢配合！
[班主任姓名]
```

## 使用提示
- 群通知优先短句，不堆背景解释
- 涉及时间、地点、材料时尽量分行写清楚
- 一条通知只保留一个主任务，避免混杂多个事项
