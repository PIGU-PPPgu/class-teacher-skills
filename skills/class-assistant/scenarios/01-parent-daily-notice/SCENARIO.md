# 场景 01｜家长群日常通知

## 场景目标
快速生成适合家长群发送的日常通知，包括着装提醒、集合时间、材料准备、作业提醒、普通班务通知。

## 适用时机
- 明天有升旗、活动、调课、集合等需要统一通知时
- 需要把老师口头信息整理成清晰文案时
- 需要在正式版、简洁版之间切换时

不适合：
- 涉及处分、冲突、隐私问题的个别沟通
- 需要单独私聊某位家长的敏感情况

## 输入
### 必填
- 事项名称
- 时间
- 执行动作 / 要求

### 可选
- 地点
- 对象
- 截止时间
- 注意事项
- 语气偏好（正式 / 简洁 / 温和）

### 缺失时默认处理
- 未给地点：不强行补充
- 未给语气：默认“正式、简洁”
- 未给对象：默认按班级家长群口径输出

## 输出
默认输出：
- 1 条可直接复制的家长群通知

可选输出：
- 更正式版
- 更简洁版
- 需要回执确认版

## 推荐语气
- 正式
- 简洁
- 温和
- 明确

## 文件说明
- `templates.md`：通知类标准模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 帮我写一个明天升旗穿校服的家长群通知
- 帮我把这段通知改得更简洁正式
- 帮我出一个需要家长回复“收到”的版本

## 注意事项
- 默认先生成草稿，不直接外发
- 一条通知尽量只说一件主事
- 涉及截止时间时，尽量前置
- 不包含真实学生隐私与联系方式
