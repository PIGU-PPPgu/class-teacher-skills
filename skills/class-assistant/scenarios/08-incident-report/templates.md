# 模板｜突发事件上报 / 情况说明

## 模板 1｜标准情况说明版
**适用：** 内部记录、年级组说明  
**语气：** 客观、审慎

```markdown
## 情况说明

**时间：** [时间]
**地点：** [地点]
**涉及人员：** [涉及人员]

### 一、事件经过
[客观描述经过]

### 二、现场处理情况
- [处理措施 1]
- [处理措施 2]

### 三、当前状态
[当前状态]

### 四、后续跟进
- [后续动作 1]
- [后续动作 2]
```

## 模板 2｜极简留痕版
**适用：** 先记录，后续补充  
**语气：** 简短

```markdown
- 时间：[时间]
- 地点：[地点]
- 事件：[经过简述]
- 已处理：[处理措施]
- 待跟进：[后续事项]
```

## 使用提示
- 先写事件经过，再写处理动作
- 如果信息未核实，明确标注“据现场了解”或“初步情况”
- 不要把情绪评价写进事实记录
