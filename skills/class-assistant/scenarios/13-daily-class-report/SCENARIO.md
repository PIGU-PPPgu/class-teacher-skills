# 场景 13｜班级日报

## 场景目标
把一天内的班级出勤、学习、纪律、活动和异常情况整理成一份简洁日报，方便老师留痕、汇报或交接。

## 适用时机
- 需要记录当天班级整体运行情况
- 需要向年级组、搭班老师或自己留档汇总
- 需要把零散事件整理成结构化记录
- 需要形成后续跟进依据
- 不适合替代正式事故报告或处分材料

## 输入
- 必填信息：日期、班级、当天重点情况
- 可选信息：出勤、作业、纪律、活动、家校沟通、异常事项
- 缺失时默认：按“基本情况—重点事件—后续跟进”输出

## 输出
- 默认输出：一版内部班级日报
- 可选输出：精简汇报版、交接版、周报素材版
- 输出对象：班主任、搭班老师、内部记录

## 推荐语气
- 客观
- 简洁
- 条理清楚
- 不情绪化
- 便于追踪

## 文件说明
- `templates.md`：日报结构模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入自动汇总脚本

## 直接给 AI 的说法
- 请把今天班级情况整理成一版内部日报。
- 请根据这些零散记录，按出勤、纪律、学习、跟进事项归类。
- 请生成一版适合和搭班老师交接的班级日报。

## 注意事项
- 默认用于内部，不直接外发家长群
- 记录以事实为主，不写夸张判断
- 涉及敏感学生情况时，控制披露范围
