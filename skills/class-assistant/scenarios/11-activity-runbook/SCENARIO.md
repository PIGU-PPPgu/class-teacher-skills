# 场景 11｜活动流程单

## 场景目标
把一次班级活动的时间线、分工、物料与提醒事项整理成一份清晰可执行的流程单，方便老师现场推进与复盘。

## 适用时机
- 班级外出活动、校内主题活动、家长开放日等前期准备
- 需要多人协作、分工明确的班级活动
- 需要提前检查时间节点与物料清单
- 活动当天需要一页式执行稿
- 不适合替代学校正式安全审批文件

## 输入
- 必填信息：活动名称、日期时间、地点、主要环节
- 可选信息：参与人员、分工、物料、风险提醒、集合要求
- 缺失时默认：先按“准备—进行—收尾”输出基础流程

## 输出
- 默认输出：活动执行流程单
- 可选输出：老师内部版、对家长通知版、学生简版须知
- 输出对象：老师、协作同事、家长或学生

## 推荐语气
- 清楚
- 可执行
- 条理化
- 不空泛
- 提前预警

## 文件说明
- `templates.md`：活动流程与分工模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入签到/分组辅助脚本

## 直接给 AI 的说法
- 请把下面活动安排整理成老师现场可用的流程单。
- 请根据这次春游信息生成一版“时间—事项—负责人”流程表。
- 请把活动准备、当天流程和收尾工作拆开写清楚。

## 注意事项
- 默认先给执行稿，不默认外发
- 涉及校外活动时，提醒补充安全要求与应急联系人
- 不要遗漏集合时间、负责人和收尾动作
