# 模板｜活动流程单

## 模板 1｜标准活动流程单
**适用：** 班级活动、主题活动、开放日  
**语气：** 清楚、执行导向  
**可替换字段：**
- [活动名称]
- [日期时间]
- [地点]
- [参与对象]
- [流程节点]
- [负责人]

```text
活动名称：[活动名称]
日期时间：[日期时间]
活动地点：[地点]
参与对象：[参与对象]

一、活动前准备
- [准备事项 1]｜负责人：[负责人]
- [准备事项 2]｜负责人：[负责人]
- [准备事项 3]｜负责人：[负责人]

二、活动当天流程
- [时间 1]｜[环节 1]｜负责人：[负责人]
- [时间 2]｜[环节 2]｜负责人：[负责人]
- [时间 3]｜[环节 3]｜负责人：[负责人]

三、物料检查
- [物料 1]
- [物料 2]
- [物料 3]

四、风险提醒
- [风险点 1]
- [风险点 2]
- [应急处理方式]

五、活动后收尾
- [收尾事项 1]
- [收尾事项 2]
- [反馈或复盘安排]
```

- [时间]

- [负责人]

```text
[时间] - [事项] - [负责人]
[时间] - [事项] - [负责人]

```
