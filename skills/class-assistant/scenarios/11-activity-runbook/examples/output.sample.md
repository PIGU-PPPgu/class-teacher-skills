# 示例输出｜活动流程单

活动名称：四年级春季劳动实践
日期时间：4 月 30 日 14:00-16:30
活动地点：学校劳动基地
参与对象：四年级二班全体学生

一、活动前准备
- 13:40 前清点人数并整队｜负责人：班主任
- 检查劳动工具与手套｜负责人：科任老师
- 讲清安全要求与分组安排｜负责人：班主任

二、活动当天流程
- 14:00｜统一集合，讲解流程与安全要求｜负责人：班主任
- 14:15｜分组进入实践区域｜负责人：科任老师
- 15:30｜各组展示劳动成果｜负责人：班主任
- 16:00｜整理工具、清点人数、返班｜负责人：全体带队老师

三、风险提醒
- 工具使用时不得打闹
- 分组活动不擅自离队
- 返程前再次核对人数

四、活动后收尾
- 回班进行简短总结
- 记录活动照片与表现亮点
