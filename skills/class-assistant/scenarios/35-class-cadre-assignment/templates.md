# 模板｜班干部岗位分配

## 模板 1｜标准岗位分配表
**适用：** 新学期班干部安排、阶段性调整  
**语气：** 清楚、执行导向  

```text
班干部岗位安排：

班长：[姓名]
副班长：[姓名]
学习委员：[姓名]
纪律委员：[姓名]
卫生委员：[姓名]
劳动委员：[姓名]
宣传委员：[姓名]
文体委员：[姓名]
小组长：[姓名 / 分组填写]

备注：
- [备注1]
- [备注2]
```

## 模板 2｜主岗 / 副岗建议版
**适用：** 需要岗位备份与培养梯队  
**语气：** 简洁、可执行

```text
班长：主岗 [姓名] / 副岗 [姓名]
学习委员：主岗 [姓名] / 副岗 [姓名]
纪律委员：主岗 [姓名] / 副岗 [姓名]
```

## 模板 3｜学生公布版
**适用：** 班内公布或家长群同步  
**语气：** 简明、正式

```text
同学们，以下为本阶段班干部岗位安排，请大家从 [开始时间] 起按新分工履职。
如有特殊情况，请课后单独和老师沟通，不要私下自行调整岗位。
```
