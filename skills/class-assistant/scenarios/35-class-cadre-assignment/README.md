# 场景 35｜班干部岗位分配

适合根据学生名单、岗位设置和班级管理要求，快速生成一版可执行的班干部分工方案。

## 这个场景解决什么问题
- 班干部岗位多，手工分配容易顾此失彼
- 需要兼顾责任心、组织力、表达力、纪律示范和学习状态
- 需要同时考虑主岗、副岗、轮岗和培养梯队
- 安排后还常常需要一版班内公布说明和老师内部备注

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：岗位分配表与说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 班干部岗位分配表
- 主岗 / 副岗建议版
- 调整说明
- 公布版 / 内部备注版

## 一句话用法
把学生名单、岗位清单和分配规则给 AI，让它直接整理出一版班干部岗位分配方案。
