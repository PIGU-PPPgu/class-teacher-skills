# 示例输出｜班干部岗位分配

## 输出 1｜岗位安排表（内部草稿）
```text
班干部岗位安排：

班长：林雪
副班长：张晨
学习委员：李可
纪律委员：陈雨
卫生委员：刘欣
宣传委员：周妍

备注：
- 班长与副班长分别承担统筹与执行推进。
- 学习委员与纪律委员均优先考虑了稳定性与示范作用。
- 卫生、宣传岗位兼顾细致度与表达能力。
```

## 输出 2｜主岗 / 副岗建议版
```text
班长：主岗 林雪 / 副岗 张晨
学习委员：主岗 李可 / 副岗 王哲
纪律委员：主岗 陈雨 / 副岗 吴凡
```

## 输出 3｜学生公布版
```text
同学们，以下为本阶段班干部岗位安排，请大家从本周起按新分工认真履职。
如有特殊情况，请课后单独和老师沟通，不要私下自行调整岗位。
```
