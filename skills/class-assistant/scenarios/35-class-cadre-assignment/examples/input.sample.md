# 示例输入｜班干部岗位分配

## 示例 1｜新学期班委安排
```text
班级人数：12 人
岗位目标：确定本阶段班干部岗位，兼顾执行力、责任心、表达能力和培养梯队

学生名单：
1. 张晨（男，组织能力强，执行力强）
2. 李可（女，认真细致，表达较好）
3. 王哲（男，学习稳定）
4. 陈雨（女，责任心强）
5. 赵宁（男，积极，但纪律需继续观察）
6. 刘欣（女，做事稳妥）
7. 孙浩（男，行动力强）
8. 周妍（女，表达力较好）
9. 吴凡（男，配合度高）
10. 郑雅（女，细心）
11. 何睿（男，安静，执行认真）
12. 林雪（女，组织协调能力强）

岗位清单：
- 班长
- 副班长
- 学习委员
- 纪律委员
- 卫生委员
- 宣传委员

额外规则：
- 核心岗位优先给责任心强、执行力稳定的学生
- 纪律委员需有示范作用
- 学习委员尽量由学习状态稳定的学生担任
- 适当保留培养梯队，不要所有强项都压在同一人身上
```
