# 场景 35｜班干部岗位分配

## 场景目标
根据学生名单、岗位设置和班级管理要求，生成清楚、均衡、便于执行的班干部岗位分配方案，帮助班主任快速完成班委搭建、职责分工与后续培养安排。

## 适用时机
- 新学期首次确定班干部
- 阶段性调整班委分工
- 需要设置主岗、副岗、轮岗或见习岗位
- 需要兼顾责任心、组织能力、表达能力、纪律示范、学习状态等因素
- 需要输出班内公布版和老师内部培养版
- 不适合替代学校正式评优评先或处分决定

## 输入
- 必填信息：学生名单、岗位清单、分配目标
- 可选信息：责任心、表达能力、组织能力、纪律表现、学习态度、上课态度、培养潜力、回避组合、已有任职经历
- 缺失时默认：优先均衡分配核心岗位，给出一版主岗建议表

## 输出
- 默认输出：一版班干部岗位分配表
- 可选输出：主岗 / 副岗建议版、轮岗安排版、培养梯队版、学生公布版、内部备注版
- 输出对象：班主任、学生、家长（如需）

## 推荐语气
- 清楚
- 公平
- 不冗长
- 执行导向
- 便于后续观察调整

## 文件说明
- `templates.md`：岗位分配表与说明模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接自动分配脚本

## 直接给 AI 的说法
- 请按这份名单给我分一版班干部岗位安排。
- 请把责任心、表达能力、纪律示范和培养潜力一起考虑进去。
- 请给我一版老师内部用安排说明，再给一版可直接对学生公布的版本。

## 注意事项
- 默认先出草稿，发布前建议老师再核对学生实际状态
- 不要在公布版中公开写学生敏感标签或负面评价
- 若岗位数量与学生能力结构不匹配，优先标注需后续培养或轮岗
