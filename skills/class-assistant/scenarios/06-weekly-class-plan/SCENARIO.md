# 场景 06｜班级周计划 / 班务提醒

## 场景目标
帮助老师把一周内的重点工作、每日安排和待办事项整理成清晰的班级周计划或班务提醒，便于执行和追踪。

## 适用时机
- 每周前统一梳理班务工作时
- 需要给自己或搭班老师整理本周重点时
- 需要把班会、值日、收缴、活动等事项排进周计划时

不适合：
- 需要精确到分钟级的复杂排班
- 替代正式校务日历系统

## 输入
### 必填
- 时间范围
- 本周重点事项

### 可选
- 每日安排
- 截止事项
- 班级活动
- 需要提醒的对象
- 输出风格（清单式 / 日历式 / 简版）

### 缺失时默认处理
- 未给每日安排：按重点事项生成周清单
- 未给风格：默认清单式
- 未给对象：默认老师自用班务提醒口径

## 输出
默认输出：
- 1 份班级周计划 / 班务提醒清单

可选输出：
- 按天拆分版
- 家长群可发送版
- 教师内部协作版

## 推荐语气
- 清楚
- 执行导向
- 简洁

## 文件说明
- `templates.md`：周计划模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 帮我整理下周班级周计划
- 按周一到周五帮我拆开
- 帮我做成可发给搭班老师的版本

## 注意事项
- 重点事项不超过 3-5 个
- 截止时间尽量写明确
- 可执行优先于好看
