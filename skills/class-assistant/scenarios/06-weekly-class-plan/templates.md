# 模板｜班级周计划 / 班务提醒

## 模板 1｜标准周计划版
**适用：** 老师自用 / 内部协作  
**语气：** 清单式、执行导向

```markdown
# 第[X]周班级工作计划

**时间：** [日期] - [日期]

## 本周重点
1. [重点事项 1]
2. [重点事项 2]
3. [重点事项 3]

## 每日安排
### 周一
- [事项]

### 周二
- [事项]

### 周三
- [事项]

### 周四
- [事项]

### 周五
- [事项]

## 截止事项
- [截止日期]：[事项]

## 备注
- [补充说明]
```

## 模板 2｜简版班务提醒
**适用：** 快速汇总本周工作  
**语气：** 简洁

```text
本周班务重点：
1. [事项 1]
2. [事项 2]
3. [事项 3]

截止提醒：
- [截止时间]前完成[事项]
```

## 使用提示
- 周计划先抓重点，再拆每日安排
- 每天事项不宜过多，避免计划失真
- 截止类任务单独列出，防止遗漏
