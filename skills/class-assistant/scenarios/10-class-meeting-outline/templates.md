# 模板｜班会提纲

## 模板 1｜常规班会提纲版
**适用：** 每周班会、常规管理  
**语气：** 清楚、稳健、简洁  
**可替换字段：**
- [班会主题]
- [班级]
- [时长]
- [近期情况]
- [重点提醒]

```text
班会主题：[班会主题]
适用班级：[班级]
时长：[时长]

一、开场导入（2-3 分钟）
- 简要说明本次班会主题
- 结合近期情况引出重点：[近期情况]

二、主体内容（10-12 分钟）
- 重点 1：[重点提醒 1]
- 重点 2：[重点提醒 2]
- 重点 3：[重点提醒 3]

三、互动环节（3-5 分钟）
- 提问 / 讨论：[互动问题]
- 学生自查：[自查任务]

四、总结与要求（2-3 分钟）
- 重申关键要求
- 布置后续落实事项：[后续任务]
```

## 模板 2｜专题教育班会版
**适用：** 安全、纪律、学习习惯等专题  
**语气：** 引导、克制、有重点  
**可替换字段：**
- [专题主题]
- [案例]
- [讨论问题]
- [行动建议]

```text
班会主题：[专题主题]

一、导入
从[案例]切入，说明为什么今天要讨论这个问题。

二、问题展开
1. 目前常见情况是什么
2. 为什么会出现这些问题
3. 可能带来的影响是什么

三、讨论互动
请同学围绕以下问题思考：
- [讨论问题 1]
- [讨论问题 2]

四、行动建议
- [行动建议 1]
- [行动建议 2]
- [行动建议 3]

五、结尾
强调今天最重要的一句话，并明确接下来一周的落实要求。
```
