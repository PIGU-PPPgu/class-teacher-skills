# 场景 10｜班会提纲

## 场景目标
把班会主题、目标、流程和话术整理成一份可直接使用的班会提纲，帮助老师快速备课与现场组织。

## 适用时机
- 每周常规班会准备
- 围绕纪律、安全、学习习惯、心理状态做专题班会
- 节假日前后的提醒班会
- 需要先有结构化提纲，再决定是否扩写讲稿
- 不适合替代涉及专业心理危机干预的正式流程

## 输入
- 必填信息：班会主题、年级/班级、时长
- 可选信息：近期班级问题、希望强调的重点、互动环节、结尾任务
- 缺失时默认：按“开场—主题—互动—总结”生成基础结构

## 输出
- 默认输出：一版可直接照着讲的班会提纲
- 可选输出：精简版、详细主持版、板书要点版
- 输出对象：班主任本人

## 推荐语气
- 清楚
- 稳定
- 有引导感
- 不说教过度
- 有鼓励性

## 文件说明
- `templates.md`：常规班会与专题班会模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入提纲生成辅助脚本

## 直接给 AI 的说法
- 请根据下面主题，生成一个 20 分钟班会提纲。
- 请把这个纪律教育主题整理成“开场—案例—讨论—总结”的班会结构。
- 请给我一版适合小学高年级的安全教育班会提纲，语气不要太硬。

## 注意事项
- 默认先给提纲，不直接扩成整篇讲稿
- 避免空泛口号，尽量贴近班级真实情境
- 涉及敏感个案时，不在班会上公开细节
