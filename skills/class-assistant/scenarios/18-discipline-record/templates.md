# 模板｜违纪情况记录

## 模板 1｜标准内部记录版
**适用：** 班主任留档、后续跟进  
**语气：** 客观、克制  
**可替换字段：**
- [时间]
- [地点]
- [学生/对象]
- [事实经过]
- [现场处理]
- [后续安排]

```text
【情况记录】
时间：[时间]
地点：[地点]
涉及对象：[学生/对象]

一、事实经过
[事实经过]

二、现场处理
[现场处理]

三、后续安排
[后续安排]
```

## 模板 2｜家长沟通前底稿
**适用：** 与家长沟通前整理事实  
**语气：** 平实、明确  
**可替换字段：**
- [学生姓名]
- [事件]
- [已处理情况]
- [需要家长配合事项]

```text
今天在[事件]中，[学生姓名]出现了相关不当行为。现场已进行了及时处理和提醒。
后续希望家长协助孩子一起重视[需要家长配合事项]，我们也会继续跟进。
```
