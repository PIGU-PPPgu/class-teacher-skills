# 场景 18｜违纪情况记录

## 场景目标
把学生违纪或常规问题整理成一份客观、可留痕、便于后续跟进的记录稿，帮助老师减少情绪化表述。

## 适用时机
- 课间冲突、课堂违纪、屡次提醒无效后需要记录
- 需要向年级组或家长做事实说明前，先整理内部稿
- 需要形成后续谈话、教育或回访依据
- 需要区分事实经过、已采取措施和后续安排
- 不适合写成定性过度、带指控色彩的处分材料

## 输入
- 必填信息：时间、地点、涉及对象、具体行为、现场处理
- 可选信息：相关教师反馈、学生说明、后续约定、家校沟通安排
- 缺失时默认：只写已确认事实，不补写动机推断

## 输出
- 默认输出：内部记录版
- 可选输出：家长沟通前说明底稿、年级组同步版
- 输出对象：老师内部使用为主

## 推荐语气
- 客观
- 克制
- 明确
- 去情绪化

## 文件说明
- `templates.md`：违纪记录与后续跟进模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续有批量记录脚本，可放这里

## 直接给 AI 的说法
- 帮我把这次课堂违纪整理成客观记录，不要写成指责口吻。
- 根据下面信息，生成一版内部留档用的违纪情况记录。
- 把这件事分成“事实经过—现场处理—后续安排”来写。

## 注意事项
- 默认先生成草稿，不直接外发
- 不进行人格化评价，不贴标签
- 涉及冲突、伤害、严重安全问题时，保留人工复核
