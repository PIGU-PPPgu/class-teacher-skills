# 场景 40｜班级编排方案（分组 / 座位 / 值日 / 班委一体化）

## 场景目标
根据学生名单和班级管理理念，一次性生成分组、座位、值日、班委岗位四位一体的班级编排方案，帮助班主任在新学期或新阶段快速搭建完整的班级运行结构。

## 适用时机
- 新学期第一次全面编排（分组、座位、值日、班委一步到位）
- 阶段性大调整（需要同时重排分组和座位）
- 班级引入积分 / 货币 / 游戏化管理模式时
- 已有分组或班委框架，需要补充缺失环节
- 不适合替代学校正式分班或行政决定

## 输入
- 必填信息：学生名单（含性别）、班级总人数、编排目标
- 可选信息：
  - 分组相关：每组人数、男女搭配要求、成绩层次、回避同组名单
  - 座位相关：教室行列结构、座位主题风格（如大富翁地图式）、与积分联动规则
  - 值日相关：值日周期、值日岗位清单、轮值规则
  - 班委相关：岗位设置、人数配置、职责范围、薪资 / 积分标准
  - 管理体系：积分规则、货币体系、奖惩标准
- 缺失时默认：按人数均衡、男女搭配、4人一组输出基础编排方案

## 输出
- 默认输出：一版完整的班级编排方案（含分组表、座位图、值日表、班委岗位表）
- 可选输出：
  - 分组 + 组内编号版
  - 座位地图标注版（含区域说明）
  - 值日轮换表（按天 / 按周）
  - 班委职责与薪资 / 积分标准版
  - 积分与座位联动规则说明版
  - 学生公布版 vs 内部备注版
- 输出对象：班主任内部使用、班内公布、家长同步（如需）

## 推荐语气
- 清楚
- 结构化
- 执行导向
- 不冗长
- 便于后续微调

## 文件说明
- `templates.md`：分组表、座位图、值日表、班委岗位表、积分规则模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 请根据这份 48 人的名单，帮我一次性排好分组（12 组，每组 4 人）、座位（大富翁主题）、值日轮换和班委岗位。
- 我想用一个主题化的管理方式，比如"大富翁"式座位，积分最高的小组可以优先选座。请帮我设计完整方案。
- 我已经有分组了，请帮我补充座位编排、值日表和班委职责体系，要有积分薪资标准。

## 注意事项
- 默认先出草稿，发布前建议老师再核对实际情况
- 不要在公布版中公开写学生敏感标签或负面评价
- 所有学生姓名必须脱敏处理后再作为示例使用
- 若编排规则冲突，优先显式标注无法同时满足的条件
- 积分 / 货币体系数值仅为参考，老师可根据班情自行调整
