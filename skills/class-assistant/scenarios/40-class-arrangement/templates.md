# 模板｜班级编排方案

## 模板 1｜分组表（带组内编号）
**适用：** 新学期建组、阶段调整分组  
**语气：** 清楚、执行导向  
**可替换字段：**
- [组号]
- [组员1姓名]～[组员N姓名]
- [性别]
- [组内编号]

```text
分组表（共 [X] 组）：

第1组：[组员1]（[性别]，1号）、[组员2]（[性别]，2号）、[组员3]（[性别]，3号）、[组员4]（[性别]，4号）
第2组：[组员1]（[性别]，1号）、[组员2]（[性别]，2号）、[组员3]（[性别]，3号）、[组员4]（[性别]，4号）
...
第[X]组：[组员列表]

编排规则：
- 每组 [N] 人，男女尽量均衡
- 组内编号按 [排序规则] 编排
- [其他规则]
```

## 模板 2｜主题座位编排（大富翁地图式）
**适用：** 引入游戏化管理的主题座位方案  
**语气：** 清楚、有趣、执行导向  
**可替换字段：**
- [区域名称]
- [区域颜色]
- [对应小组]

```text
座位编排——[主题名称]地图式

教室平面图（讲台在上方）：

┌──────────────────────────────────────────┐
│                  讲台                      │
├──────────────────────────────────────────┤
│ [区域1]  │ [区域2]  │ [区域3]  │ [区域4]  │
│ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │
│ 第X组    │ 第X组    │ 第X组    │ 第X组    │
├──────────┼──────────┼──────────┼──────────┤
│ [区域5]  │ [区域6]  │ [区域7]  │ [区域8]  │
│ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │
│ 第X组    │ 第X组    │ 第X组    │ 第X组    │
├──────────┼──────────┼──────────┼──────────┤
│ [区域9]  │ [区域10] │ [区域11] │ [区域12] │
│ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │ 蓝色地产 │
│ 第X组    │ 第X组    │ 第X组    │ 第X组    │
└──────────┴──────────┴──────────┴──────────┘

区域说明：
- 蓝色地产（可购买/可选区域）：积分排名靠前的小组优先选座
- 绿色/黄色区域（政府/老师区域）：讲台旁、特殊功能位
- 红色区域（公摊走廊）：过道位、临时位

选座规则：
1. 每月/每阶段按小组积分排名顺序选座
2. 积分最高小组优先选蓝色地产区域
3. 选座后该阶段内固定，下次积分更新后重新选座
```

## 模板 3｜值日轮换表
**适用：** 按组编排卫生值日、值日班长轮流  
**语气：** 简洁、可执行  
**可替换字段：**
- [周次]
- [组号]
- [值日班长姓名]

```text
值日轮换表

一、值日班长轮流制（每天 1 人）：

周一：[值日班长1]（第X组）
周二：[值日班长2]（第X组）
周三：[值日班长3]（第X组）
周四：[值日班长4]（第X组）
周五：[值日班长5]（第X组）
（按组内编号依次轮换，每周自动顺延）

二、卫生值日表（按组编排）：

第 [N] 周：第[X]组负责
- 扫地：[组员1]
- 拖地：[组员2]
- 擦黑板：[组员3]
- 倒垃圾：[组员4]
- 整理桌椅：[全员]

岗位分工（可组内轮换）：
- 卫生委员（监督）：[姓名]
- 纪检委（纪律检查）：[姓名]
- 统计局长（学习/积分统计）：[姓名]
```

## 模板 4｜班委职责与薪资/积分标准表
**适用：** 班委岗位分配、职责说明、积分薪资体系  
**语气：** 清楚、正式  
**可替换字段：**
- [岗位名称]
- [人数]
- [职责描述]
- [薪资/积分标准]

```text
班委职责体系

| 岗位名称       | 人数 | 主要职责                     | 薪资/积分    |
|----------------|------|------------------------------|-------------|
| 银行行长       | 2人  | 管理班级货币出入账           | [X]/周      |
| 司法局长(班长) | 2人  | 判罚纠纷、维持秩序           | [X]/周      |
| 纪检委会长     | 1人  | 监督纪律(值日班长轮值)       | [X]/天      |
| 卫生管理局长   | 1人  | 监督卫生、检查值日           | [X]/周      |
| 统计局长(学习) | 1人  | 收集统计积分、学习数据       | [X]/周      |
| 教育局长(课代) | N人  | 作业收发登记、打分授权       | [X]/周      |
| 文化旅游局长   | 2人  | 班级活动策划、文化建设       | [X]/次      |
| 监狱狱长       | 2人  | 违纪跟进、督促整改           | [X]/周      |

课代表体系（每科 1 人）：
- 语文课代表：[姓名]
- 数学课代表：[姓名]
- 英语课代表：[姓名]
- [其他科目]课代表：[姓名]

课代表权限：
- 负责本科作业收发和登记
- 有打分授权，可直接给作业表现加减 [X] 分
```

## 模板 5｜学生公布版（综合编排通知）
**适用：** 班内公布完整编排方案  
**语气：** 简明、正式、鼓励参与  

```text
同学们：

本阶段班级编排方案如下，请从 [开始时间] 起按新安排执行。

一、分组安排
[分组表摘要]

二、座位安排
座位采用 [主题名称] 地图式编排，积分排名靠前的小组优先选座。
具体座位请查看教室座位图。

三、值日安排
值日班长每天轮换，卫生值日按组轮值。
详细排班请查看值日表。

四、班委与岗位
本阶段班委及岗位安排如下：
[班委岗位表摘要]

如有疑问或特殊情况，请课后单独和老师说明。
```
