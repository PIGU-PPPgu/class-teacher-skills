# 示例输入｜班级编排方案

## 示例 1｜新学期全面编排（48人班级，大富翁主题）

```text
班级人数：48人
编排目标：一次性完成分组、座位、值日、班委四位一体编排

一、分组要求
- 分成12组，前11组每组4人，第12组可放剩余人数
- 男女搭配，每组尽量2男2女
- 每位组员有组内编号（1～4号或1～6号）
- 需要兼顾成绩层次和性格互补

学生名单（已脱敏）：
第1～12人：
1. 陈明远（男，成绩较好，性格外向）
2. 李思琪（女，成绩中上，组织力强）
3. 张浩然（男，成绩中等，纪律需关注）
4. 王雨萱（女，成绩较好，性格内向）
5. 刘子轩（男，成绩中，体育突出）
6. 赵雅婷（女，成绩中上，表达力强）
7. 孙博文（男，成绩较好，责任心强）
8. 周欣怡（女，成绩中，合作力好）
9. 吴俊杰（男，成绩中，纪律需关注）
10. 郑诗涵（女，成绩较好，细致认真）
11. 何睿泽（男，成绩中下，性格内向）
12. 林雅琪（女，成绩中等，动手能力强）

（第13～48人信息类似，省略）

二、座位要求
- 教室结构：3大排，每排4个区域
- 采用大富翁地图式座位：每个区域对应一块"地产"
- 颜色分区：蓝色（可购买/可选）、绿色黄色（老师区域）、红色（公摊走廊）
- 座位与积分联动：积分最高小组优先选座

三、值日要求
- 值日班长每天1人轮流
- 卫生值日按组轮值
- 需要岗位：卫生委员、纪检委、统计局长（学习委员）

四、班委要求
- 银行行长2人：管理班级货币
- 纪检委会长1人：监督纪律（值日班长）
- 司法局长(班长)2人：判罚纠纷
- 卫生管理局长1人：监督卫生
- 统计局长(学习委员)1人：统计积分
- 教育局长(7科课代表)：作业管理，有打分授权
- 文化旅游局长2人：活动策划
- 监狱狱长2人：违纪跟进

五、积分/薪资体系
- 各岗位有周薪或日薪（班级货币）
- 积分规则需要体现奖惩
```
