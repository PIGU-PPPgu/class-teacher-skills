# 模板｜成绩分析结论输出

## 模板 1｜班级整体分析版
**适用：** 班级考试整体复盘  
**语气：** 客观、清晰

```markdown
## 班级成绩分析

### 一、整体情况
本次考试班级整体表现为：[整体判断]。
平均分：[平均分]
及格率：[及格率]
优秀率：[优秀率]

### 二、主要特点
1. [特点 1]
2. [特点 2]
3. [特点 3]

### 三、进步与问题
- 进步方面：[进步结论]
- 问题方面：[问题结论]

### 四、后续建议
- [建议 1]
- [建议 2]
- [建议 3]
```

## 模板 2｜重点学生关注版
**适用：** 识别需跟进学生  
**语气：** 内部工作稿

```markdown
## 重点关注学生
- [学生 A]：问题表现 / 变化情况 / 建议动作
- [学生 B]：问题表现 / 变化情况 / 建议动作
- [学生 C]：问题表现 / 变化情况 / 建议动作
```

## 使用提示
- 先写整体，再写个体
- 建议部分尽量对应前面的问题
- 没有统计值时不要自行编造数字
