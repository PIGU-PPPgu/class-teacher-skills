# 场景 04｜成绩分析结论输出

## 场景目标
把原始成绩、排名、进步退步情况整理成老师能快速阅读的结论稿，便于班级复盘、重点学生识别、家长会准备与后续跟进。

## 适用时机
- 一次考试后需要快速形成班级分析
- 需要找出进步最大、退步明显、重点关注学生时
- 需要把数据结论转换成文字报告时

不适合：
- 数据明显缺失或口径不一致但仍强行下结论
- 用公开文本暴露学生隐私
- 用一次成绩给学生贴长期标签

## 输入
### 必填
- 成绩数据 / 汇总结果
- 分析目标（班级整体 / 重点学生 / 进退步 / 学科对比）

### 可选
- 班级平均分
- 及格率 / 优秀率
- 上次考试对比数据
- 重点关注名单
- 输出风格（简版 / 标准 / 汇报版）

### 缺失时默认处理
- 没有对比数据：只做本次横向结论
- 没有统计指标：以已有数据为准，不虚构数字
- 没有风格要求：默认标准结论稿

## 输出
默认输出：
- 1 份成绩分析结论稿

可选输出：
- 班级整体总结
- 重点学生名单与建议
- 家长会可用版本
- 年级组汇报版

## 推荐语气
- 客观
- 聚焦结论
- 少空话
- 可执行

## 文件说明
- `templates.md`：成绩分析结论模板
- `examples/`：脱敏输入输出样例

## 直接给 AI 的说法
- 帮我把这次月考成绩整理成班级分析
- 找出这次考试重点关注学生
- 帮我出一个家长会能用的成绩总结

## 注意事项
- 结论与数据要一致
- 不把偶发成绩波动直接上升为能力定性
- 涉及个人数据时优先脱敏
