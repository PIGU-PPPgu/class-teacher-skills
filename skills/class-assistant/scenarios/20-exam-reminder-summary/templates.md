# 模板｜考前提醒 / 考后总结

## 模板 1｜考前提醒版
**适用：** 考试前一天或当周提醒  
**语气：** 稳定、明确  
**可替换字段：**
- [考试类型]
- [时间]
- [准备事项]
- [提醒重点]

```text
各位同学/家长好：
[时间]将进行[考试类型]。请大家提前做好以下准备：
- [准备事项 1]
- [准备事项 2]
- [准备事项 3]

请重点留意：[提醒重点]。保持正常作息，认真准备即可。
```

## 模板 2｜考后总结版
**适用：** 班级整体考试后简要复盘  
**语气：** 客观、鼓励  
**可替换字段：**
- [考试类型]
- [整体表现]
- [做得较好的方面]
- [需要改进的方面]
- [后续安排]

```text
关于本次[考试类型]，班级整体表现为：[整体表现]。
做得较好的方面是：[做得较好的方面]；仍需改进的是：[需要改进的方面]。
接下来我们会继续围绕[后续安排]做好调整和跟进。
```
