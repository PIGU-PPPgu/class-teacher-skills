# 场景 36｜小组任务分工 / 课堂分工安排

## 场景目标
根据小组名单、课堂任务和班级管理要求，生成清楚、均衡、便于执行的小组任务分工方案，帮助班主任或任课老师快速明确组内角色与责任。

## 适用时机
- 课堂合作学习任务分工
- 项目式学习、小组展示、主题活动分工
- 需要明确组长、记录员、发言人、材料员、纪律提醒员等角色
- 需要兼顾学生表达力、执行力、合作度、学习态度和上课态度
- 需要输出学生执行版和老师内部观察版
- 不适合替代正式学业评价结论

## 输入
- 必填信息：小组名单、任务内容、分工目标
- 可选信息：表达能力、组织能力、执行力、合作态度、学习态度、上课态度、回避搭配、轮岗要求
- 缺失时默认：按组内角色均衡分配，给出一版基础分工表

## 输出
- 默认输出：一版小组任务分工表
- 可选输出：角色安排建议版、轮岗版、学生执行版、老师内部备注版、观察重点摘要
- 输出对象：班主任、任课老师、学生、家长（如需）

## 推荐语气
- 清楚
- 公平
- 不冗长
- 执行导向
- 方便课堂直接使用

## 文件说明
- `templates.md`：分工表与说明模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接自动分工脚本

## 直接给 AI 的说法
- 请按这份小组名单给我出一版课堂任务分工。
- 请把表达力、执行力、合作度和学习态度一起考虑进去。
- 请给我一版老师内部用说明，再给一版可直接发给学生执行的版本。

## 注意事项
- 默认先出草稿，发布前建议老师再核对课堂目标与学生实际状态
- 不要在公布版中公开写学生敏感标签或负面评价
- 若组内人数与角色数量不匹配，优先标注兼岗或轮岗安排
