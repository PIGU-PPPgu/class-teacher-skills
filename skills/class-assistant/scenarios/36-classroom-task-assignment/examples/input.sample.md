# 示例输入｜小组任务分工 / 课堂分工安排

## 示例 1｜课堂合作任务
```text
任务主题：小组完成一次语文阅读展示
目标：每组明确角色，避免分工不清

第1组成员：张晨、李可、王哲、陈雨、赵宁
第2组成员：刘欣、孙浩、周妍、吴凡、郑雅
第3组成员：何睿、林雪、周航、许诺、彭然

分工目标：
- 每组都要有组长、发言人、记录员、材料员、纪律提醒员
- 组长优先安排组织力较强的学生
- 发言人优先安排表达较好的学生
- 记录员优先安排细致认真的学生
- 纪律提醒员优先安排规则意识较强的学生

额外规则：
- 尽量让经常沉默的学生承担一次可控表达任务
- 不要让同一个学生长期固定在单一角色
```
