# 模板｜小组任务分工 / 课堂分工安排

## 模板 1｜标准分工表
**适用：** 课堂合作任务、项目活动分工  
**语气：** 清楚、执行导向  

```text
第1组：
- 组长：[姓名]
- 发言人：[姓名]
- 记录员：[姓名]
- 材料员：[姓名]
- 纪律提醒员：[姓名]

第2组：
- 组长：[姓名]
- 发言人：[姓名]
- 记录员：[姓名]
- 材料员：[姓名]
- 纪律提醒员：[姓名]
```

## 模板 2｜老师内部说明版
**适用：** 记录分工逻辑和课堂观察重点  
**语气：** 简洁、可执行

```text
分工说明：
- [组别] 由 [姓名] 担任组长，主要考虑其组织推进能力。
- [组别] 由 [姓名] 担任发言人，主要考虑其表达与展示能力。
- [组别] 观察重点：[观察重点]
```

## 模板 3｜学生执行版
**适用：** 班内公布或课堂直接执行  
**语气：** 简明、正式

```text
同学们，以下为本次课堂任务的小组分工，请大家按分工完成合作任务。
如有特殊情况，请及时向老师反馈，不要自行更换角色。
```
