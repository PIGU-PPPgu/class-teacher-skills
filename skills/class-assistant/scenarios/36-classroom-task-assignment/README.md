# 场景 36｜小组任务分工 / 课堂分工安排

适合根据小组名单、任务类型和课堂目标，快速生成一版可执行的小组任务分工方案。

## 这个场景解决什么问题
- 小组合作时，任务常常分工不清，容易出现有人闲、有人全包
- 需要兼顾表达、记录、组织、执行和展示等不同能力
- 需要区分组长、发言人、记录员、材料员等不同角色
- 分工后还需要一版老师内部说明和学生执行版

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：分工表与说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 小组任务分工表
- 角色安排建议版
- 课堂执行说明
- 公布版 / 内部备注版

## 一句话用法
把小组名单、任务内容和分工规则给 AI，让它直接整理出一版小组任务分工方案。
