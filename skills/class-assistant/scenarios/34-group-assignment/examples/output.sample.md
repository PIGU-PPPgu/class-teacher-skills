# 示例输出｜分组编排 / 学习小组分配

## 输出 1｜分组表（内部草稿）
```text
分组表：

第1组：张晨｜刘欣｜赵宁｜郑雅
第2组：陈雨｜王哲｜何睿｜周妍
第3组：孙浩｜李可｜吴凡｜林雪

备注：
- 每组都尽量配置了 1 名带动能力较强的学生。
- 纪律需关注学生未集中在同一组。
- 男女搭配整体较均衡。
```

## 输出 2｜组长建议版
```text
第1组：建议组长 张晨
- 组内分工：张晨负责统筹，刘欣负责记录，赵宁负责执行事项，郑雅负责汇报。

第2组：建议组长 陈雨
- 组内分工：陈雨负责统筹，王哲负责资料整理，何睿负责补充意见，周妍负责展示。

第3组：建议组长 林雪
- 组内分工：林雪负责统筹，孙浩负责推进，李可负责记录，吴凡负责汇报。
```

## 输出 3｜学生公布版
```text
同学们，以下为本阶段新的学习小组安排，请从明天起按新分组开展课堂合作与小组任务。
如有特殊情况，请课后单独和老师说明，不要自行换组。
```
