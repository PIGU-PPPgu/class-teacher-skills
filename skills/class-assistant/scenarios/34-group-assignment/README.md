# 场景 34｜分组编排 / 学习小组分配

适合根据学生名单、分组人数和班级规则，快速生成一版可执行的小组分配方案。

## 这个场景解决什么问题
- 手工分组容易只顾一个维度，忽略整体平衡
- 需要兼顾学习水平、性格、纪律、男女搭配和重点学生分散
- 需要输出老师内部版和学生公布版
- 分组后还常常需要组长、组名、座区或任务分工建议

## 建议先看
- `SCENARIO.md`：场景目标、输入输出、边界
- `templates.md`：标准分组表与说明模板
- `examples/input.sample.md`：示例输入
- `examples/output.sample.md`：示例输出

## 典型输出
- 学习小组分组表
- 组长建议版
- 分组说明
- 公布版 / 内部备注版

## 一句话用法
把学生名单、每组人数和分组规则给 AI，让它直接整理出一版学习小组分配方案。
