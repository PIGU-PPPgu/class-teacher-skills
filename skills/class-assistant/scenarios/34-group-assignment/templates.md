# 模板｜分组编排 / 学习小组分配

## 模板 1｜标准分组表
**适用：** 常规学习小组、活动小组初稿  
**语气：** 清楚、执行导向  
**可替换字段：**
- [第1组]
- [第2组]
- [第3组]
- [第4组]
- [第5组]
- [第6组]
- [备注]

```text
分组表：

第1组：[第1组]
第2组：[第2组]
第3组：[第3组]
第4组：[第4组]
第5组：[第5组]
第6组：[第6组]

备注：
- [备注1]
- [备注2]
```

## 模板 2｜组长与分工建议版
**适用：** 需要同步给出组长和组内任务建议  
**语气：** 简洁、可执行

```text
第1组：组长 [姓名]
- 组内分工：[分工建议]

第2组：组长 [姓名]
- 组内分工：[分工建议]
```

## 模板 3｜学生公布版
**适用：** 班内公布或发群通知  
**语气：** 简明、正式

```text
同学们，以下为本阶段学习小组安排，请从 [开始时间] 起按新分组开展活动。
如有特殊情况，请课后单独和老师说明，不要自行换组。
```
