# 模板｜班级周报

## 模板 1｜标准内部周报版
**适用：** 班主任周复盘、内部汇总  
**语气：** 客观、概括、清楚  
**可替换字段：**
- [周次]
- [班级]
- [总体情况]
- [学习情况]
- [纪律情况]
- [活动亮点]
- [家校沟通]
- [下周重点]

```text
【班级周报】
周次：[周次]
班级：[班级]

一、本周总体情况
[总体情况]

二、学习情况
[学习情况]

三、纪律与常规
[纪律情况]

四、活动与亮点
[活动亮点]

五、家校沟通
[家校沟通]

六、下周重点
[下周重点]
```

## 模板 2｜复盘简版
**适用：** 自我复盘、快速回顾  
**语气：** 精炼、结论导向  
**可替换字段：**
- [本周最好]
- [本周问题]
- [下周动作]

```text
本周最好的一点：[本周最好]
本周最需关注的问题：[本周问题]
下周优先动作：[下周动作]
```
