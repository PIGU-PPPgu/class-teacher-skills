# 模板｜家长会发言提纲

## 模板 1｜标准家长会提纲
**适用：** 5 分钟左右发言  
**语气：** 稳重、清晰

```markdown
## 家长会发言提纲

### 一、开场
- 感谢家长到场
- 简要说明本次家长会主题

### 二、班级整体情况
- 学习状态：[整体情况]
- 行为习惯：[整体情况]
- 班级氛围：[整体情况]

### 三、当前主要问题
1. [问题 1]
2. [问题 2]
3. [问题 3]

### 四、需要家长配合的重点
- [配合建议 1]
- [配合建议 2]
- [配合建议 3]

### 五、后续工作计划
- [计划 1]
- [计划 2]

### 六、结束语
- 强调家校协同
- 表达期待与感谢
```

## 使用提示
- 控制信息密度，避免变成冗长汇报
- 建议部分尽量落到家庭作息、阅读、作业、情绪沟通等可执行事项
