# 场景 15｜班级月报

## 场景目标
把一个月内班级运行情况整理成一份结构化月报，帮助老师回看整体趋势、沉淀重点问题，并为下月工作安排提供依据。

## 适用时机
- 每月结束后做阶段性总结
- 需要从周报与零散记录中提炼月度结论
- 需要向内部做阶段汇总或作为家长会素材准备
- 需要沉淀持续问题与进步趋势
- 不适合直接替代正式考核材料或对外宣传稿

## 输入
- 必填信息：月份、班级、本月总体情况
- 可选信息：学习趋势、出勤情况、纪律表现、活动亮点、家校沟通、需持续关注事项、下月重点
- 缺失时默认：按“总体—亮点—问题—下月计划”输出

## 输出
- 默认输出：一版内部班级月报
- 可选输出：精简汇报版、家长会素材版、阶段复盘版
- 输出对象：班主任、内部协作老师

## 推荐语气
- 客观
- 稳定
- 有趋势判断
- 不夸张
- 便于规划下月行动

## 文件说明
- `templates.md`：月报结构模板
- `examples/`：脱敏输入输出样例
- `scripts/`：如后续需要可接入周报汇总脚本

## 直接给 AI 的说法
- 请把本月班级情况整理成一版内部月报。
- 请根据这几周的记录，提炼本月进步点、持续问题和下月重点。
- 请生成一版适合家长会前自己参考的班级月度总结。

## 注意事项
- 月报重点是趋势与判断，不是周报简单拼接
- 不默认公开学生个体细节
- 涉及敏感事件时，以概括性表述为主
